## 🔧 技术实现

### 1. 模板数据文件
- **文件位置**：`desktop-app-vue/src/renderer/data/defaultTemplates.js`
- **导出格式**：ES6 Module (`export const defaultTemplates`)
- **数据结构**：
  ```javascript
  {
    id: string,              // 模板唯一ID
    name: string,            // 模板名称
    description: string,     // 模板描述
    project_type: string,    // 项目类型
    is_builtin: boolean,     // 是否内置模板
    usage_count: number,     // 使用次数（模拟数据）
    config_json: string,     // 配置JSON字符串
    subtitle?: string,       // 可选：副标题
    author?: string,         // 可选：作者
    date?: string,          // 可选：日期
  }
  ```

### 2. 集成方式
- 在 `ProjectsPage.vue` 中导入：`import { defaultTemplates } from '@/data/defaultTemplates'`
- 通过 `getDefaultTemplates()` 函数返回模板数组
- 当后端API加载失败时，自动使用默认模板

### 3. 展示逻辑
- 项目页面下方展示模板区域
- 每页显示8个模板（`.slice(0, 8)`）
- 点击模板打开详情弹窗
- 支持"做同款"功能，一键创建项目

## 🎯 使用说明

### 用户端
1. 进入项目页面
2. 向下滚动查看"项目模板"区域
3. 点击任意模板卡片查看详情
4. 点击"做同款"按钮开始创建项目
5. AI自动拆解任务并开始执行

### 开发端
```javascript
// 导入模板数据
import { defaultTemplates } from '@/data/defaultTemplates';

// 获取所有模板
const allTemplates = defaultTemplates;

// 按分类筛选
const writingTemplates = defaultTemplates.filter(t => t.project_type === 'write');
const pptTemplates = defaultTemplates.filter(t => t.project_type === 'ppt');

// 按category筛选（从config_json解析）
const selfMediaTemplates = defaultTemplates.filter(t => {
  const config = JSON.parse(t.config_json);
  return config.category === '自媒体创作';
});
```
