# 常见问题

## 产品相关

### 什么是ChainlessChain?

ChainlessChain是一个完全去中心化的个人AI管理系统，整合了知识库管理、去中心化社交和交易辅助三大核心功能，通过U盾和SIMKey提供军事级安全保护。

### ChainlessChain的主要特点是什么?

- 🔐 **军事级安全**: 基于U盾/SIMKey的硬件级加密
- 🌐 **完全去中心化**: 数据完全由用户掌控
- 🧠 **AI原生**: 本地大模型，隐私与智能兼得
- 📱 **跨设备协作**: PC、手机无缝同步
- 🔓 **开源自主**: 代码透明，数据自己掌控

### ChainlessChain是免费的吗?

核心功能完全免费：
- ✅ 知识库管理（无限容量）
- ✅ 去中心化社交
- ✅ 小额交易辅助
- ✅ 本地AI模型

可选收费服务：
- 云端LLM API额度
- 托管Git仓库（超过免费额度）
- 高级功能和企业版

### 支持哪些平台?

- ✅ **PC端**: Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+)
- ✅ **移动端**: Android 9+, iOS 13+ (部分功能)
- ⏳ **Web端**: 即将推出

## 安全相关

### 我的数据安全吗?

非常安全：
- 🔐 硬件级密钥保护（U盾/SIMKey）
- 🔒 AES-256加密存储
- 🔏 端到端加密通信
- 🛡️ 开源代码可审计
- 📝 完整的隐私保护机制

### 如果丢失U盾/SIMKey怎么办?

建议提前备份：
1. **助记词备份**: 24个单词，抄写保存
2. **备份U盾**: 制作备份U盾存放在安全地点
3. **云端加密备份**: 使用强密码加密后备份到云端

如果有备份，可以恢复所有数据。没有备份则无法恢复。

### 开发团队能看到我的数据吗?

不能。ChainlessChain采用端到端加密：
- ❌ 开发团队无法访问你的数据
- ❌ 没有后门
- ❌ 没有主密钥
- ✅ 只有你自己能解密数据

3. **RAG配置**: 合理的检索和重排序

建议：
- 重要问题使用云端API

- 定期完善知识库

## 故障排查

### 无法启动应用

检查清单：
1. 确认系统版本符合要求
2. 检查是否安装了所有依赖
3. 查看错误日志：`~/.chainlesschain/logs/`
4. 尝试重新安装
5. 联系技术支持

### U盾/SIMKey无法识别

**U盾**:
1. 检查是否插入USB接口
2. 重新安装驱动
3. 尝试其他USB接口
4. 在设备管理器中查看

**SIMKey**:
1. 确认SIM卡已插入
2. 检查SIM卡是否为USIM卡
3. 授予应用权限
4. 重启手机

### 同步失败

常见原因：
1. **网络问题**: 检查网络连接
2. **Token过期**: 重新生成Token
3. **仓库冲突**: 手动解决冲突
4. **存储空间不足**: 清理空间

### AI问答没有结果

可能原因：
1. **知识库为空**: 先添加内容
2. **模型未加载**: 检查模型状态
3. **搜索词不匹配**: 换个问法
4. **相似度阈值太高**: 降低阈值
