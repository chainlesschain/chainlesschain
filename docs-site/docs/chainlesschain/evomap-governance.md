# IP & Governance DAO (Phase 77)

## Overview

Gene IP protection and decentralized governance for the EvoMap ecosystem.

## Features

- DID+VC originality proof
- Anti-plagiarism detection
- Derivation chain tracking
- Revenue split management
- Governance proposals and voting

## IPC Channels

| Channel                               | Description                |
| ------------------------------------- | -------------------------- |
| `evomap-gov:register-ownership`       | Register gene ownership    |
| `evomap-gov:trace-contributions`      | Trace contributions        |
| `evomap-gov:create-proposal`          | Create governance proposal |
| `evomap-gov:cast-vote`                | Cast governance vote       |
| `evomap-gov:get-governance-dashboard` | Get governance dashboard   |

## Database Tables

- `gene_ownership` — Ownership records
- `evomap_governance_proposals` — Proposal tracking
