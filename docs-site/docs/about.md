# 关于我们

## 我们的使命

**用技术捍卫隐私，用AI赋能个人**

在这个数据被视为"新石油"的时代，我们相信每个人都应该拥有自己数据的完全控制权。ChainlessChain致力于构建一个去中心化的个人AI管理系统，让每个人都能安全地管理知识、社交和交易，无需依赖任何中心化平台。

## 我们的故事

### 起源

ChainlessChain项目始于2023年，创始团队在使用各种笔记应用时发现：
- 📝 数据存储在别人的服务器上
- 🔓 隐私无法得到保障
- 💰 功能受限需要付费
- 🔒 平台倒闭数据就丢失

我们认为这不应该是未来的样子。

### 愿景

我们设想一个世界：
- 每个人完全掌控自己的数据
- AI助手运行在你自己的设备上
- 社交不需要中心化平台
- 交易不需要中介抽成
- 知识是真正属于你的资产

## 常见误解

### Q: ChainlessChain是区块链项目吗?

A: 不完全是。我们利用区块链技术（智能合约、去中心化存储），但不是纯粹的区块链项目。核心是AI知识管理系统。

### Q: 需要持有加密货币吗?

A: 不需要。核心功能完全免费，不需要任何加密货币。只有在使用交易辅助功能时，可选择使用加密货币托管。

### Q: 数据存在哪里?

A: 数据存储在你自己的设备上（PC、手机），可选择同步到Git仓库（已加密）。我们的服务器不存储用户数据。
