## 📐 尺寸规范

### 二维码
- **尺寸**：200 x 200 像素
- **格式**：PNG
- **容错级别**：高（30%）
- **颜色**：黑色前景 + 白色背景

### Open Graph
- **尺寸**：1200 x 630 像素
- **格式**：PNG或JPG
- **文件大小**：<1MB
- **安全区域**：四周留白50px

### 产品截图
- **尺寸**：800 x 600 像素（或16:9比例）
- **格式**：PNG
- **压缩**：TinyPNG压缩后<200KB

### Logo
- **尺寸**：200 x 60 像素（或更大，按比例缩放）
- **格式**：PNG（透明背景）
- **备用格式**：SVG（矢量，可无限缩放）

### 徽章
- **尺寸**：200 x 80 像素
- **格式**：PNG
- **背景**：可透明或纯色

## 🔧 图片优化

### 压缩工具

**在线工具：**
1. **TinyPNG**（推荐）
   - 网址：https://tinypng.com/
   - 优点：智能压缩，质量几乎无损
   - 限制：每次最多20张，每张5MB

2. **Squoosh**
   - 网址：https://squoosh.app/
   - 优点：开源，支持多种格式
   - 特点：可调整压缩参数

**命令行工具：**
```bash
# ImageMagick（批量压缩）
convert input.png -quality 85 output.png

# pngquant（PNG专用）
pngquant --quality=65-80 input.png
```

### 格式选择

| 图片类型 | 推荐格式 | 原因 |
|---------|---------|------|
| Logo | PNG/SVG | 需要透明背景/矢量 |
| 二维码 | PNG | 清晰度要求高 |
| 产品截图 | PNG/JPG | PNG质量好，JPG文件小 |
| OG分享图 | PNG/JPG | 社交平台支持 |
| 徽章 | PNG | 可能需要透明背景 |

## 🆘 常见问题

### Q1：二维码扫描不出来？

**原因**：
- 二维码太小
- 打印/显示模糊
- 容错级别太低

**解决**：
- 确保至少200x200像素
- 使用高容错级别（H级30%）
- 测试：用手机扫描电脑屏幕上的二维码

### Q2：Logo在深色背景下看不清？

**解决**：
1. 制作两个版本：
   - logo-light.png（浅色Logo，用于深色背景）
   - logo-dark.png（深色Logo，用于浅色背景）
2. 或使用SVG，CSS控制颜色

### Q3：产品截图太大，加载慢？

**解决**：
```bash
# 使用TinyPNG压缩
# 或调整尺寸
convert input.png -resize 800x600 -quality 85 output.png
```

### Q4：Python脚本报错？

**常见错误及解决：**
