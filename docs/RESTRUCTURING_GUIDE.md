## 📝 文档格式建议

每个文档应包含:

1. **标题和简介**

   ```markdown
   # 文档标题

   简要说明文档内容和用途
   ```

2. **目录** (对于长文档)

   ```markdown
   ## 目录

   - [章节1](#章节1)
   - [章节2](#章节2)
   ```

3. **内容章节**
   - 使用清晰的标题层级
   - 添加代码示例
   - 包含必要的链接

4. **相关文档链接**

   ```markdown
   ## 相关文档

   - [功能详解](./FEATURES.md)
   - [安装指南](./INSTALLATION.md)
   ```
