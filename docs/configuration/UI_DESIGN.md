# ChainlessChain UI/UX 设计规范

## 设计理念

**简洁 · 高效 · 安全**

- **简洁**: 信息层级清晰，减少认知负担
- **高效**: 快捷操作，提升工作效率
- **安全**: 强调隐私保护，给用户信任感

## 组件规范

### 1. 按钮 (Button)

**尺寸:**
- Small: 24px height
- Medium: 32px height (默认)
- Large: 40px height

**类型:**
```jsx
<Button type="primary">主要按钮</Button>
<Button type="default">次要按钮</Button>
<Button type="dashed">虚线按钮</Button>
<Button type="text">文字按钮</Button>
<Button type="link">链接按钮</Button>
<Button danger>危险按钮</Button>
```

### 2. 输入框 (Input)

```jsx
<Input placeholder="搜索笔记..." prefix={<SearchOutlined />} />
<Input.TextArea rows={4} placeholder="输入笔记内容..." />
<Input.Password placeholder="请输入PIN码" />
```

### 3. 卡片 (Card)

```jsx
<Card
  title="笔记标题"
  extra={<Button type="link">编辑</Button>}
  hoverable
>
  笔记内容摘要...
</Card>
```

### 4. 列表 (List)

```jsx
<List
  dataSource={knowledgeItems}
  renderItem={item => (
    <List.Item
      actions={[<Button type="link">编辑</Button>]}
    >
      <List.Item.Meta
        avatar={<FileTextOutlined />}
        title={item.title}
        description={item.created_at}
      />
    </List.Item>
  )}
/>
```

## 交互规范

### 1. 动画时长

```css
/* 快速: 小型元素 */
--duration-fast: 0.1s;

/* 正常: 中型元素 */
--duration-normal: 0.2s;

/* 慢速: 大型元素 */
--duration-slow: 0.3s;
```

### 2. 缓动函数

```css
--ease-in: cubic-bezier(0.4, 0, 1, 1);
--ease-out: cubic-bezier(0, 0, 0.2, 1);
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
```

### 3. 状态反馈

**加载状态:**
- 使用Spin组件显示加载动画
- 超过3秒显示进度条
- 提供取消操作选项

**错误状态:**
- 使用Message/Notification提示错误
- 提供重试按钮
- 记录错误日志

**成功状态:**
- 使用简短的成功提示
- 自动关闭 (3秒)
- 必要时提供撤销选项

## 响应式设计

### 断点
```css
/* 小屏幕 */
--screen-sm: 768px;

/* 中等屏幕 */
--screen-md: 992px;

/* 大屏幕 */
--screen-lg: 1200px;

/* 超大屏幕 */
--screen-xl: 1600px;
```

### 自适应布局
- 宽度 < 768px: 单栏布局，隐藏侧边栏
- 宽度 768px - 1200px: 双栏布局
- 宽度 > 1200px: 完整三栏布局

## 可访问性 (A11y)

### 1. 键盘导航
- 所有交互元素可通过Tab键访问
- 使用合理的tabIndex顺序
- 支持Escape关闭弹窗

### 2. 屏幕阅读器
- 为图标添加aria-label
- 使用语义化HTML标签
- 提供alt文本

### 3. 对比度
- 文本与背景对比度 ≥ 4.5:1
- 大文本对比度 ≥ 3:1

## 性能优化

### 1. 虚拟滚动
- 列表超过100项时使用虚拟滚动
- 使用react-window或react-virtualized

### 2. 懒加载
- 图片懒加载
- 路由懒加载
- 组件按需加载

### 3. 防抖节流
- 搜索输入: 300ms防抖
- 窗口resize: 200ms节流
- 滚动事件: 100ms节流

## 用户体验细节

### 1. 空状态
- 提供友好的空状态提示
- 引导用户创建第一条笔记
- 显示示例数据

### 2. 错误处理
- 友好的错误提示
- 提供解决方案
- 允许重试操作

### 3. 加载状态
- 骨架屏占位
- 进度指示器
- 取消长时间操作

### 4. 过渡动画
- 页面切换使用淡入淡出
- 列表项添加/删除使用滑动动画
- 弹窗使用缩放动画
