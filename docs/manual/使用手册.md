## 第一章 系统概述

### 1.1 产品简介

ChainlessChain（无链之链）是一套面向党政机关、事业单位和大中型企业的一体化人工智能办公系统。它将以下能力整合在同一个平台中：

- **AI 智能助手**：辅助起草文档、整理资料、回答问题
- **统一知识库**：管理本单位的所有文档、笔记、会议纪要
- **加密即时通信**：替代微信、钉钉等做工作沟通
- **多端协同办公**：电脑、手机、命令行多端数据同步
- **公文与决策协同**：电子公文流转、多人多签

所有数据存储在用户单位自己的设备和服务器上，不上传任何第三方。

### 1.2 三种使用形态

ChainlessChain 提供三种产品形态，对应不同使用场景：

| 形态 | 适合 | 主要用途 |
|---|---|---|
| **桌面端** | 办公室主力工作站 | 日常办公、文档编辑、AI 对话、深度协作 |
| **移动端** | 出差、外勤、随身携带 | 拍照入库、语音记录、远程审批、扫码 |
| **命令行** | 服务器、自动化、技术人员 | 后台任务、批量处理、脚本集成 |

三种形态共用同一份数据，任意一端的修改自动同步到其它端。

### 1.3 主要功能模块

| 模块 | 简要说明 |
|---|---|
| 智能助手 | AI 对话、文档起草、检索问答、技能调用 |
| 知识库 | 文档管理、笔记、检索、项目空间 |
| 即时通信 | 一对一、群组、频道，端到端加密 |
| 公文协同 | 公文起草、审批、签发、归档 |
| 会议管理 | 会议安排、纪要自动生成 |
| 多签决策 | 重大事项多人签名后生效 |
| 跨单位通信 | 与其它单位建立加密通信通道 |
| 数字资产管理 | 多链钱包、跨链桥（可选模块）|
| 系统管理 | 用户、角色、权限、审计 |

### 1.4 关键特性

- **数据主权**：所有数据在客户单位自己的设备和服务器中
- **硬件密钥**：身份由 U盾、SIM 卡硬件密钥保护
- **端到端加密**：通信内容只有发送方与接收方可解密
- **断网可用**：本地大模型支持下整套系统断网可运行
- **信创合规**：全栈适配国产 CPU / 操作系统 / 数据库 / 大模型
- **完整审计**：所有关键操作留痕，可追溯

## 第二章 安装与首次配置

### 2.1 安装前准备

#### 2.1.1 确认硬件与系统

请先确认您的设备满足系统要求：

- **桌面端**：详见 [附录 D 系统要求](#附录-d-系统要求)
- **移动端**：Android 8.0 及以上
- **命令行**：可在任何能运行 Node.js 22 的服务器上使用

#### 2.1.2 准备硬件密钥

如果您单位采购了 U盾（USB 硬件密钥），请准备好。首次启动时需要插入。

如果还没有 U盾，可以先用密码方式使用，后续添加 U盾。

#### 2.1.3 获取安装包

向单位的系统管理员索取安装包，或访问单位指定的内部下载地址。

注意：请勿从非官方渠道下载安装包。

### 2.2 桌面端安装

#### 2.2.1 Windows 安装

【操作步骤】

1. 双击安装包 `ChainlessChain-Setup-x.y.z.exe`
2. 如果系统弹出"用户账户控制"提示，点击 `是`
3. 选择安装语言（默认简体中文），点击 `下一步`
4. 阅读并同意《最终用户许可协议》
5. 选择安装位置（建议保留默认）
6. 选择是否创建桌面快捷方式
7. 点击 `安装`，等待安装完成（约 3-5 分钟）
8. 安装完成后点击 `完成`，软件自动启动

【预期结果】

桌面出现 ChainlessChain 图标，开始菜单中出现快捷方式。

【注意事项】

- 安装过程中如果 Windows Defender 提示拦截，选择 `允许`
- 如果安装失败，请暂时关闭杀毒软件后重试
- 不建议安装到 C 盘根目录或包含中文 / 空格的路径

#### 2.2.2 麒麟 / 统信 UOS 安装

【操作步骤】

1. 打开终端，进入安装包所在目录
2. 执行命令：`sudo dpkg -i chainlesschain_x.y.z_amd64.deb`
3. 如果提示缺少依赖，执行：`sudo apt-get install -f`
4. 安装完成后，从应用菜单中找到 ChainlessChain 启动

【预期结果】

应用菜单中出现 ChainlessChain，可正常启动。

#### 2.2.3 macOS 安装（适用于支持的硬件）

【操作步骤】

1. 双击 `ChainlessChain-x.y.z.dmg` 挂载安装包
2. 将 ChainlessChain 图标拖到 `Applications` 文件夹
3. 在 Launchpad 中找到 ChainlessChain 启动
4. 首次启动如系统提示"无法验证开发者"，请进入 `系统偏好设置 → 安全性与隐私`，点击 `仍要打开`

### 2.3 移动端安装

#### 2.3.1 Android 安装

【操作步骤】

1. 在手机上打开安装包 `ChainlessChain-x.y.z.apk`
2. 如果系统提示"未知来源应用"，请进入 `设置 → 安全`，开启 `允许来自此来源`
3. 点击 `安装`
4. 安装完成后，桌面出现应用图标
5. 点击图标启动

【注意事项】

- 必须使用单位指定渠道下载的安装包
- 不同手机品牌的"未知来源"开启路径可能略有不同，请按提示操作

### 2.4 命令行工具安装

#### 2.4.1 前置条件

需要在服务器上预先安装 Node.js 22 LTS 版本。

【操作步骤】

1. 打开终端
2. 执行：`npm install -g chainlesschain`
3. 等待下载与安装（约 2 分钟）
4. 验证安装：`cc -v`
5. 如果显示版本号（如 `0.161.12`），说明安装成功

【预期结果】

可在任意目录使用 `cc` 命令。

### 2.5 首次启动配置向导

无论桌面端、移动端还是命令行，首次启动都会进入配置向导。请按以下步骤完成配置。

#### 2.5.1 选择界面语言

【操作步骤】

1. 启动应用
2. 在语言选择界面，选择 `简体中文`（默认）
3. 点击 `下一步`

#### 2.5.2 创建或导入身份

每个用户需要有一个数字身份用于登录。如果您是首次使用，请创建新身份；如果已经有身份（在其它设备上用过），请选择导入。

【创建新身份】

1. 选择 `创建新身份`
2. 输入显示名称（如"张三"，可改）
3. 系统生成 12 个或 24 个汉字 / 英文单词组成的"恢复短语"
4. **极其重要**：将恢复短语**抄写到纸上**，妥善保存
   - 不要拍照
   - 不要发到任何聊天工具
   - 不要保存到电脑、邮箱、网盘
   - 这是您身份的唯一备份，丢失则无法恢复
5. 系统要求您按顺序点选恢复短语，验证您已正确备份
6. 完成后系统在本设备生成密钥并加密保存

【导入已有身份】

1. 选择 `导入已有身份`
2. 输入 12 / 24 词恢复短语
3. 系统在本设备恢复您的身份

#### 2.5.3 设置解锁方式

【操作步骤】

1. 设置 6 位以上 PIN（建议 8 位以上含数字、字母）
2. 如果设备支持指纹 / 面容识别，可选择启用
3. 如果有 U盾，可选择"使用 U盾解锁"

【注意事项】

- PIN 不要与微信、银行卡相同
- 启用生物识别后，下次解锁更便捷
- 启用 U盾后，必须插入 U盾才能解锁，安全等级最高

#### 2.5.4 配置 AI 助手

【本地 AI 模式（推荐党政场景）】

1. 选择 `本地 AI`
2. 系统检测是否已安装 Ollama 等本地 AI 服务
3. 选择您要使用的模型（推荐：通义千问 7B / 文心 / DeepSeek 7B）
4. 如果模型未下载，点击 `下载模型`，等待完成（约 5-30 分钟，视模型大小与网络）

【远端 AI 模式】

如果您单位有远端 AI 服务（如内部部署的大模型 API），可填入：

1. 选择 `远端 AI`
2. 填写服务地址（管理员提供）
3. 填写认证凭据
4. 点击 `测试连接`

#### 2.5.5 连接单位协同服务（如有）

如果您单位部署了 ChainlessChain 协同服务器，可在此连接，实现与同事的数据同步与通信。

【操作步骤】

1. 选择 `加入单位协同`
2. 输入单位协同服务器地址（管理员提供，如 `chainlesschain.your-org.local`）
3. 输入单位邀请码（管理员提供）
4. 等待管理员审批通过

如果暂不连接，可后续在 `设置 → 单位协同` 中配置。

#### 2.5.6 完成配置

点击 `完成配置`，进入主界面。

#### 3.1.1 侧边栏图标说明

- **用户头像**：当前用户信息、切换身份、退出登录

### 3.2 智能助手使用

#### 3.2.1 开始一次对话

【操作步骤】

1. 点击侧边栏 `💬 智能助手`
2. 点击 `+ 新建对话`
3. 在底部输入框中输入您的问题
4. 按 `Enter` 发送（按 `Shift + Enter` 换行）
5. AI 会逐字流式回复

【预期结果】

AI 在右侧聊天区域逐字显示回答。

【提示】

- 提问越具体，回答越准确
- 可以多轮追问，AI 会记住上下文
- 不满意的回答可以点 `重新生成`

#### 3.2.2 让 AI 读您的文档回答

【适用场景】

- 询问会议纪要中的某个决议
- 让 AI 总结某份长报告
- 对比多份合同的差异

【操作步骤】

1. 在新建对话或对话中，点击输入框上方的 `引用资料` 按钮
2. 选择要让 AI 阅读的文档（可多选）
3. 输入您的问题
4. 发送

【示例】

引用 `2026年4月部门会议纪要.docx` + `2026年5月部门会议纪要.docx`，提问：

> 这两个月会议中，关于 X 项目的进展有哪些不同？

AI 会基于这两份文档回答，并标注引用的具体段落。

#### 3.2.3 让 AI 起草文档

【适用场景】

- 起草通知 / 报告 / 请示
- 写邮件回复
- 整理会议纪要

【操作步骤】

1. 在对话中描述您想要的文档
2. 例如：

> 帮我起草一份关于"加强办公电脑安全管理"的通知，发给全机关，500 字左右，措辞庄重。

3. AI 起草后，您可以：
   - 直接复制使用
   - 让 AI 修改（如"把第二段改得更具体"）
   - 一键导出为 Word 文档

#### 3.2.4 调用专业技能

系统内置 139 项专业技能。在输入框输入 `/` 会弹出技能列表。

【常用技能示例】

| 技能 | 用法 |
|---|---|
| `/文档转换` | 把 Word 转 PDF、PPT 转图片等 |
| `/表格处理` | Excel 数据分析、生成图表 |
| `/PPT 起草` | 根据要点生成 PPT 大纲与内容 |
| `/语音转写` | 录音文件转文字 |
| `/图片文字识别` | 拍照 / 截图中的文字提取 |
| `/合同审查` | 检查合同关键条款（律所适用） |

【操作步骤】

1. 输入 `/` 等待技能列表出现
2. 输入技能关键字筛选（如输入"PPT"）
3. 选择技能，按提示操作

#### 3.2.5 多角色 AI 协同

【适用场景】

复杂任务需要多个 AI 角色配合，例如：撰写完整调研报告、起草大型方案。

【操作步骤】

1. 主菜单 → `多角色协同`
2. 点击 `+ 新建任务`
3. 输入任务目标，例如：

> 为我们机关写一份"信创办公系统采购方案"完整调研报告，包含背景分析、市场对比、推荐方案、预算建议四部分。

4. 系统自动按 5 个阶段执行：
   - 阶段 1：理解任务
   - 阶段 2：制定执行计划
   - 阶段 3：分多个 AI 角色并行起草
   - 阶段 4：自动审核
   - 阶段 5：汇总输出
5. 每个阶段您可以：
   - 查看进度
   - 暂停 / 继续
   - 修改方向
   - 接管手动调整

【完成时间】

视任务复杂度，通常 5-30 分钟。

### 3.3 知识库使用

#### 3.3.1 添加文档

【操作步骤 - 拖拽方式】

1. 点击侧边栏 `📚 知识库`
2. 把文件从文件管理器拖到 ChainlessChain 窗口
3. 系统自动导入并提取内容

【操作步骤 - 点击导入】

1. 知识库 → 点击 `+ 新增` → `导入文件`
2. 选择文件
3. 选择归属项目（可选）
4. 选择标签（可选）
5. 点击 `确定`

【支持的文件类型】

- 文档：Word（.docx）、PDF、Markdown、纯文本
- 表格：Excel
- 演示：PPT
- 图片：JPG / PNG / GIF（自动 OCR 识别文字）
- 音频：MP3 / WAV（可选自动转写）

#### 3.3.2 写笔记

【操作步骤】

1. 知识库 → 点击 `+ 新增` → `新建笔记`
2. 在编辑器中写内容（支持 Markdown 格式）
3. 右上角设置：
   - 标题
   - 标签
   - 归属项目
4. 自动保存（无需手动点击保存）

#### 3.3.3 检索

【关键词检索】

1. 顶栏搜索框输入关键词
2. 系统返回包含该关键词的所有文档
3. 点击结果跳转到原文

### 4.3 配对桌面电脑

【适用场景】

【操作步骤】

1. **桌面端**：打开 ChainlessChain → `设置` → `移动端配对` → `生成配对二维码`

【验证】

手机首页出现"已连接桌面"卡片，显示电脑名称。

【注意事项】

- 二维码 5 分钟内有效，过期需重新生成
- 配对过程必须双方在场操作
- 解除配对：双方任一端 → `设置` → `已配对设备` → 点击对方 → `解除`

### 4.4 远程查看 / 操控桌面

   - **远程打开终端**：点 `打开远程终端`
3. 远程终端中可输入任何命令，如同在桌面端操作

【安全提示】

## 第六章 多端协同

### 6.1 数据同步原理

ChainlessChain 三端共享同一份数据。当您在桌面端写笔记后，几秒内手机和命令行端都能看到。

同步通过三种通道之一完成：

| 通道 | 适合 | 速度 |
|---|---|---|
| **局域网直连** | 同一办公室内的设备 | 最快（< 1 秒） |
| **加密 P2P** | 跨地域、外出 | 较快（1-5 秒） |
| **单位协同服务器** | 通过单位中心同步 | 取决于服务器配置 |

### 6.2 添加新设备

【适用场景】

- 新换电脑
- 在家电脑也想用
- 添加平板

【操作步骤 - 用恢复短语】

1. 新设备：安装 ChainlessChain
2. 启动后选择 `导入已有身份`
3. 输入您的 12 / 24 词恢复短语
4. 完成后新设备即可访问您的数据
5. 数据按需求自动同步

【操作步骤 - 用配对方式】

1. 老设备：`设置` → `多设备` → `添加新设备` → 显示二维码
2. 新设备：扫码 → 验证 → 完成

### 6.3 选择性同步

【适用场景】

不希望所有设备都有所有数据。例如：

- 工作电脑：所有项目
- 家用电脑：仅"个人成长"项目
- 手机：仅最近 30 天数据

【操作步骤】

1. 任一设备 → `设置` → `同步设置`
2. 在 `同步范围` 中选择：
   - 全部数据
   - 仅指定项目
   - 仅最近 N 天
   - 自定义规则

### 6.4 解决同步冲突

【冲突情景】

同一份文件您在两个设备同时修改，且修改不重叠：

- 系统自动合并
- 您不会感觉有冲突

如果修改有重叠：

- 系统标记冲突
- 通知您选择保留哪个版本
- 在 `设置` → `同步` → `待解决冲突` 中处理

## 第七章 安全与权限

### 7.1 身份与认证

#### 7.1.1 三种登录方式

| 方式 | 安全等级 | 适合 |
|---|---|---|
| PIN | 中 | 个人办公场景 |
| 生物识别（指纹 / 面容） | 中高 | 移动端 |
| U盾 / 硬件密钥 | 高 | 涉密、关键岗位 |

可同时启用多种，登录时按等级要求验证。

#### 7.1.2 修改 PIN

1. `设置` → `安全` → `修改 PIN`
2. 输入旧 PIN
3. 输入新 PIN（两次）

#### 7.1.3 添加 / 移除 U盾

1. `设置` → `安全` → `硬件密钥管理`
2. 点 `添加 U盾`，按提示插入
3. 完成后该 U盾绑定到您的身份

#### 7.1.4 启用 / 关闭生物识别

1. `设置` → `安全` → `生物识别`
2. 开关切换

### 7.2 数据加密

ChainlessChain 自动加密所有敏感数据：

- 数据库：AES-256 加密
- 通信：端到端加密
- 文件：可选启用文件级加密

**用户无需任何操作**，加密自动完成。

### 7.3 自动锁屏

【操作步骤】

1. `设置` → `安全` → `自动锁屏`
2. 设置无操作时间（默认 5 分钟）
3. 锁屏后需重新输入 PIN / 生物识别 / U盾解锁

### 7.4 审计日志

您的所有重要操作都有记录：

- 登录 / 退出
- 文件修改
- 公文签批
- 多签操作

【查看自己的操作记录】

1. `我的` → `操作历史`
2. 按时间、类型筛选

### 7.5 数据备份

#### 7.5.1 自动备份

系统自动每日全量备份，每小时增量备份。备份位置：

- 单机部署：本机 `备份` 目录
- 单位协同：单位备份服务器

#### 7.5.2 手动备份

1. `设置` → `数据` → `立即备份`
2. 选择备份位置（本机 / 外置硬盘 / 单位备份服务器）
3. 设置备份密码（恢复时需要）
4. 等待完成

#### 7.5.3 恢复备份

1. `设置` → `数据` → `从备份恢复`
2. 选择备份文件
3. 输入备份密码
4. 确认恢复（**会覆盖现有数据，请慎重**）

系统预置 5 种角色，可自定义新角色：

| 角色 | 权限 |
|---|---|
| 系统管理员 | 全部 |

| 普通用户 | 个人空间 + 部门空间 |

4. 保存

1. 管理控制台 → `用户管理` → 选择用户
2. 点击 `禁用` / `启用`

#### 8.2.4 用户离职处理

1. 管理控制台 → `用户管理` → 选择离职用户 → `离职流程`
2. 系统自动：
   - 转移用户负责的公文
   - 转移用户负责的项目
   - 备份用户个人数据
   - 注销用户身份

## 第九章 故障处理

### 9.1 故障排查通用方法

遇到问题请按以下顺序排查：

1. **看错误提示**：系统会显示错误信息，按提示操作
2. **重启应用**：约 50% 的问题通过重启解决
3. **查看日志**：`帮助` → `查看日志`
4. **联系管理员**：本单位系统管理员通常可解决多数问题
5. **联系厂商支持**：管理员无法解决的，联系 400-1068-687

### 9.2 常见故障与处理

#### 9.2.1 软件无法启动

【可能原因 1】系统资源不足

【处理】

1. 关闭占用大量内存的其他软件
2. 重启电脑
3. 重新启动 ChainlessChain

【可能原因 2】文件损坏

【处理】

1. 卸载后重新安装
2. 安装后用恢复短语登录
3. 数据自动从云端 / 单位服务器恢复

#### 9.2.2 登录失败

【可能原因 1】PIN 输错

【处理】

1. 重新输入 PIN
2. 连续输错 5 次将锁定 30 分钟

【可能原因 2】U盾未识别

【处理】

1. 拔出 U盾再插入
2. 检查 U盾驱动是否正常（设备管理器查看）
3. 如有备用 U盾，换用备用

【可能原因 3】忘记 PIN

【处理】

1. 用恢复短语重置 PIN
2. 如果连恢复短语也丢了，请联系本单位管理员（可能需要重新发放身份）

#### 9.2.3 AI 助手无响应

【可能原因 1】本地 AI 服务未启动

【处理】

1. 检查 Ollama 服务：开始菜单搜索"Ollama"启动
2. 等待 1 分钟后再问

【可能原因 2】远端 AI 服务无法访问

【处理】

1. 检查网络连接
2. `设置` → `AI` → `测试连接`
3. 联系管理员检查 AI 服务

#### 9.2.4 同步失败

【处理】

1. 检查网络
2. `设置` → `同步` → 查看同步状态
3. 点 `立即同步` 手动触发
4. 如长期无法同步，联系管理员

#### 9.2.5 配对失败

【可能原因 1】二维码已过期

【处理】

1. 桌面端重新生成二维码
2. 手机重新扫码

【可能原因 2】不在同一网络

【处理】

1. 确保手机与桌面在同一 WiFi
2. 或确保都能访问单位协同服务器

#### 9.2.6 远程终端黑屏

【处理】

1. 关闭终端会话重新打开
2. 如果反复出现，更新到最新版本

#### 9.2.7 文件无法导入

【可能原因 1】文件格式不支持

【处理】

查看支持的文件类型，转换为支持的格式后导入

【可能原因 2】文件过大

【处理】

单文件默认最大 100 MB，更大文件请压缩或分割后导入

### 9.3 错误代码对照

| 代码 | 含义 | 处理 |
|---|---|---|
| E001 | 数据库连接失败 | 重启应用 |
| E002 | 身份验证失败 | 重新登录 |
| E003 | 权限不足 | 联系管理员授权 |
| E004 | 文件不存在 | 检查文件路径 |
| E005 | 网络异常 | 检查网络 |
| E006 | AI 服务超时 | 重试 |
| E007 | 同步冲突 | 手动解决 |
| E008 | 存储空间不足 | 清理空间 |
| E009 | 版本不兼容 | 升级到匹配版本 |
| E010 | 加密密钥错误 | 联系管理员 |

完整错误代码请查阅技术支持文档或联系厂商。

### 9.4 紧急情况处理

#### 9.4.1 怀疑设备被入侵

【立即操作】

1. 拔掉网线 / 关闭 WiFi
2. 拔掉 U盾
3. 联系本单位安全负责人
4. 不要继续使用该设备
5. 在另一台设备上立即修改身份相关凭据

#### 9.4.2 U盾丢失

【立即操作】

1. 在另一台设备上登录
2. `设置` → `安全` → `挂失 U盾`
3. 该 U盾从此无法登录
4. 联系管理员补发新 U盾

#### 9.4.3 恢复短语泄露

【立即操作】

1. 立即在另一台设备上注销当前身份
2. 创建新身份
3. 把数据迁移到新身份
4. 通知管理员
5. **从此该恢复短语永久作废**

## 第十章 常见问题（FAQ）

### 10.1 关于安装与启动

**Q：第一次安装很慢，是正常的吗？**

A：是的，首次安装包含本地 AI 模型下载，可能 10-30 分钟。后续启动很快。

**Q：能不能不联网安装？**

A：可以。向管理员索要"离线安装包"，包含所有依赖。

**Q：能在多台电脑上同时登录同一个身份吗？**

A：可以。通过添加新设备或导入恢复短语，多台电脑可同时使用同一身份，数据自动同步。

### 10.2 关于使用

**Q：AI 回答不准确怎么办？**

A：

- 提问尽量具体，避免模糊问题
- 提供更多上下文（引用相关资料）
- 如果是基于知识库的问答，确保资料已正确导入
- 试试切换到更大的模型（70B 通常比 7B 准确）

**Q：可以让 AI 帮我做哪些工作？**

A：

- 起草通知、报告、邮件
- 汇总长报告、对比多份文件
- 整理会议纪要
- 检索本单位历史资料
- 处理 Excel 数据、生成图表
- 翻译、审校
- 等等，139 项内置技能

**Q：聊天记录会被存到哪里？**

A：仅存在您本地设备和单位协同服务器（如已部署）。**绝不会上传到任何第三方**。

**Q：可以删除聊天记录吗？**

A：可以。单条 / 整个对话 / 与某人的全部记录都可删除。删除后本地立即清除，远端按设置时间内同步删除。

### 10.3 关于安全

**Q：恢复短语丢了怎么办？**

A：恢复短语是您身份的唯一备份，丢失则无法找回。请务必抄到纸上离线保存。

**Q：U盾被偷了怎么办？**

A：立即在另一台设备挂失（操作见 [9.4.2](#942-u盾丢失)），该 U盾无法再登录。

**Q：单位领导能看我的私人聊天吗？**

A：不能。端到端加密决定了即使是单位管理员也看不到聊天内容。但单位可以做"通信发生记录"审计（如某时间您与某人通信过），不能看具体内容。

**Q：厂商能看我的数据吗？**

A：不能。所有数据存在您单位自己的设备 / 服务器上，且全部加密。厂商在技术上无法访问。

### 10.4 关于同步

**Q：换手机怎么办？**

A：

1. 旧手机：`设置` → `多设备` → 添加新设备生成二维码
2. 新手机：安装 ChainlessChain → 扫码导入
3. 数据自动同步

**Q：在家也想用怎么办？**

A：

- 单位允许通过加密通道连接，则在家电脑安装应用 + 导入身份即可
- 单位仅允许内网，则只在办公室使用

**Q：两个设备同时改一份文件，会覆盖吗？**

A：不会。系统会智能合并，无法合并的会通知您手动处理。

### 10.5 关于功能

**Q：能替代微信工作沟通吗？**

A：能。所有 IM 基本能力都支持，且加密强度更高、功能更专业（公文、多签等）。

**Q：能替代钉钉 / 飞书吗？**

A：核心能力（IM、文档、知识库）覆盖。专属于钉钉飞书的某些营销 / OA 自动化功能可通过我方集成开发实现。

**Q：能替代 ChatGPT 吗？**

A：能。本地大模型（如通义千问 70B）在多数办公场景下表现接近 GPT-4。涉密 / 政务场景必须用本地模型，不能用 ChatGPT。

**Q：能像 Notion 那样写文档吗？**

A：可以。支持 Markdown 编辑，所见即所得。

### 10.6 关于服务

**Q：售后响应多快？**

A：

- 标准维保：4 小时响应
- 增强维保：1 小时响应
- 紧急情况：30 分钟响应

**Q：能定制开发吗？**

A：能。常见定制：界面定制、流程定制、与现有系统集成。请联系商务咨询。

**Q：能不能上门培训？**

A：可以。一般部署后会安排上门培训：管理员 2 天、用户 0.5 天。

**Q：合同期满后怎么办？**

A：软件已永久授权使用，合同期满影响的是维保和升级服务。可续签维保，也可不续。
