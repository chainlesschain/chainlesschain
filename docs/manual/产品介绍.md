# ChainlessChain 无链之链
## 自主可控的党政机关一体化人工智能办公系统

> 厦门无链之链科技有限公司 出品 · 信创架构 · 等保三级 · 数据不出域

---

## 一句话讲清楚我们是谁

**国产自主、私有部署、硬件密钥保护的"AI + 知识库 + 即时通信 + 数字资产"一体化办公平台。** 对标 Microsoft 365 + ChatGPT Enterprise 的进口替代方案。

数据全部存放在客户单位自己的设备和服务器上，永不上传任何境外服务器；身份用硬件钥匙（U盾 / SIMKey）保护，密码丢了也不影响安全；AI 模型可全本地化部署，整套系统断网仍能运行。

---

## 为什么党政机关需要这套系统

### 当前痛点

| 痛点 | 现状 | 风险 |
|---|---|---|
| AI 助手依赖境外 | 大量员工偷偷用 ChatGPT、Claude | 涉密文件外泄 / 违反《数据安全法》《个人信息保护法》 |
| 办公软件被卡脖子 | Microsoft 365、Google Workspace 占主流 | 服务可被随时切断 / 升级被强制 |
| 知识资产沉淀难 | 文件散落个人电脑、邮箱、群聊 | 人员流动即知识流失 / 无法形成单位智力资本 |
| 即时通信不合规 | 用微信、钉钉处理工作事务 | 数据在第三方公司服务器 / 不可审计 / 不可保密 |
| 身份认证脆弱 | 密码 + 短信验证码 | 不满足等保三级双因素强认证要求 |
| 多个系统各自为政 | OA / 邮箱 / 网盘 / IM / AI 工具各买一套 | 数据不互通 / 重复采购 / 用户疲劳 |

### 我们的解决思路

**一套系统，把"个人办公主权"还给用户单位。**

- **AI 能力本地化** — 大模型在本单位机房运行，问答内容不出网
- **数据本地化** — 所有文档、对话、记录加密存放在用户自己的设备上
- **身份硬件化** — 用 U盾 / 国产密钥卡保护身份，符合等保三级
- **通信加密化** — 单位内部 IM 端到端加密，第三方（含我们厂商）也看不到内容
- **架构信创化** — 全栈兼容国产 CPU / 操作系统 / 数据库 / 大模型

## 三大核心能力

### 一、智能办公助手

让每个员工都有一个专属 AI 助手，懂本单位的业务、读得了本单位的文件、能调用本单位的工具。

| 能力 | 说明 |
|---|---|
| **AI 智能问答** | 直接拿本单位文件库问 AI，所有引用可溯源到具体文档段落 |
| **文档自动起草** | 公文、报告、PPT、Excel 数据分析，AI 一键起草后人工修改 |
| **会议纪要自动生成** | 录音 → 自动转写 → 自动整理为标准纪要格式 |
| **跨文档对比与摘要** | 几十份文件批量摘要、找出差异点、生成专题简报 |
| **139 项内置技能** | 代码审查、合规检查、表格处理、图表生成等装完即用 |
| **多角色 AI 协同** | 复杂任务由多个 AI 角色（规划 → 执行 → 审核 → 总结）配合完成 |

### 二、知识库与协同办公

把单位的文档、笔记、对话、决策记录统一管理，形成"单位的第二大脑"。

| 能力 | 说明 |
|---|---|
| **统一知识库** | 文档、笔记、聊天记录、会议纪要全部归档到本地数据库 |
| **强大检索** | 既能按关键词查（精确匹配），又能按意思查（语义检索） |
| **项目空间** | 按项目 / 部门 / 主题划分独立空间，权限隔离 |
| **多设备同步** | 办公室电脑、笔记本、手机数据自动同步，全程不经第三方云 |
| **版本与审计** | 谁在何时修改了什么，全部可追溯 |
| **完整保留人员离职后的资产** | 知识沉淀在单位，不随人走 |

### 三、加密即时通信与决策协同

替代微信 / 钉钉 / 飞书做工作沟通，所有内容端到端加密，厂商也无法解密。

| 能力 | 说明 |
|---|---|
| **加密 IM** | 一对一、多人群、频道讨论，全部端到端加密 |
| **公文流转** | 起草 → 审批 → 签发的全流程加密留痕 |
| **多签决策** | 重大决策需要多人签名才生效，符合"三重一大"决策机制 |
| **跨单位互通** | 与其它部署本系统的单位可建立加密通信通道 |
| **离线可达** | 网络中断时仍可在内网继续使用，恢复后自动同步 |

## 适用场景

### 党政机关
- 内部公文起草、AI 辅助批注、跨处室协同
- 涉密资料整理与检索（部署在涉密内网）
- 决策研讨加密 IM，符合保密委审查要求
- 重大事项多人多签留痕，符合"三重一大"

### 事业单位 / 国企
- 各部门知识库统一管理，避免人员流动造成资产流失
- 与现有 OA / 邮箱 / 协同办公系统无缝集成
- 跨地域分支机构加密通信，不依赖商业 IM
- 重要合同 / 决策的多签流程数字化

### 行业延展
- **金融**：监管报送 AI 起草、客户资料本地化分析、不出银行内网
- **医疗**：病历智能检索、不出医院网络、符合 HIPAA 类隐私要求
- **法律**：律师事务所案卷管理、AI 辅助检索、客户保密通信
- **科研**：论文资料管理、跨课题组协作、领域知识图谱

## 部署方式

我们提供四种部署模式，任选其一：

| 模式 | 适合 | 部署周期 | 数据位置 |
|---|---|---|---|
| **单机部署** | 个人用户 / 小型办公室（< 20 人） | 当天完成 | 各自电脑 |
| **局域网部署** | 中型单位（20-500 人） | 3-5 个工作日 | 单位机房一台服务器 |
| **私有云部署** | 大型单位 / 集团（500+ 人） | 1-2 周 | 单位私有云集群 |
| **涉密内网部署** | 党政涉密场景 | 按保密委要求 | 物理隔离涉密网 |

所有部署模式均**永不向我们厂商回传任何数据**，包括使用统计。

## 客户常见疑问

**Q：和钉钉 / 飞书 / 企业微信比，最大区别是什么？**
A：钉钉飞书的数据在阿里 / 字节的服务器，您看不到也管不着，遇到调取请求他们会配合监管交出。我们的数据在您单位自己的服务器，连我们厂商都看不到。

**Q：如果厂商倒闭了怎么办？**
A：所有源码您都可以拿到（合同条款保障），数据格式开放，部署的系统可以独立继续使用，不会出现"厂商一倒、系统就废"的情况。

**Q：AI 模型也要本地部署吗？性能跟得上吗？**
A：可全本地（推荐 GPU 服务器，跑国产大模型如通义千问、文心、DeepSeek 等），也可接入合规的远端模型 API。本地部署 70B 级别大模型在主流 GPU 服务器上响应速度足够日常办公使用。

**Q：现有的 OA / 邮箱 / 文件服务器能集成吗？**
A：能。提供标准 API 接口，可对接 OA 流程、邮件系统、文件服务器、AD/LDAP 域账号、单点登录系统。

**Q：员工已经习惯了用微信、钉钉，迁移会不会很痛？**
A：界面与钉钉、飞书类似，一般员工 30 分钟可上手。我们提供完整培训和一线技术支持。

**Q：怎么收费？**
A：一次性买断（按用户数 + 服务器数授权），永续使用。可选年度维保和升级服务。无任何按次、按月、按 token 的隐藏费用。

**Q：能先试用吗？**
A：可。提供 30 天 POC（概念验证）部署，我方工程师上门或远程协助安装，免费试用全部功能。
