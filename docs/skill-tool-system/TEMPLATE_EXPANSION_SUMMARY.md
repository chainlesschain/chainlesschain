## 模板特点

### 设计理念
1. **实用性优先** - 每个模板都是真实场景需求
2. **结构化** - 使用Handlebars模板引擎，支持动态变量
3. **灵活配置** - 丰富的variables_schema支持各种场景
4. **完整性** - 包含从策划到执行的完整流程

### 核心功能
- 📝 **动态变量** - 支持条件渲染、循环、计算等
- 🎨 **分类标签** - 完善的分类和子分类系统
- ⭐ **评分系统** - 用户可以对模板评分
- 📊 **使用统计** - 追踪模板使用频率
- 🔍 **搜索功能** - 支持关键词搜索
