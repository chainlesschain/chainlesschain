## 🐛 如果出错了...

### 错误1: "文件不存在"
**原因**：路径解析有问题
**检查**：
```bash
dir C:\code\chainlesschain\data\projects
```
看看项目文件夹是否存在

---

### 错误2: "文件为空" 或 "内容长度: 0"
**原因**：AI生成失败或文件保存出错
**检查**：
1. 后端AI服务是否运行？
   ```bash
   docker ps | findstr ai-service
   ```
2. 手动打开文件看看是否有内容

---

### 错误3: "预览失败" 或 "Cannot find module"
**原因**：依赖未安装
**解决**：
```bash
cd desktop-app-vue
npm install docx-preview jsdom xlsx pptx2json --save
# 然后重启应用
npm run dev
```
