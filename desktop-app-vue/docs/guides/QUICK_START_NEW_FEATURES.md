# 新功能快速开始指南

欢迎使用ChainlessChain v0.21.0！本指南将帮助你快速上手三个新增的核心功能。

## 🎤 语音输入功能

### 快速开始

1. **在AI聊天中使用**
   ```
   1. 打开AI聊天页面
   2. 点击输入框右侧的麦克风图标 🎤
   3. 允许浏览器访问麦克风
   4. 开始说话，实时查看识别结果
   5. 点击"完成"按钮，文本自动填充到输入框
   ```

2. **在知识库编辑中使用**
   ```
   1. 打开知识库编辑页面
   2. 点击编辑器工具栏的麦克风图标
   3. 录音并识别
   4. 识别文本自动追加到编辑器
   ```

### 功能特性

- ✅ 实时语音识别
- ✅ 暂停/继续录音
- ✅ 音量可视化
- ✅ 支持中文和英文

### 使用技巧

💡 **提高识别准确度**:
- 在安静的环境中录音
- 说话清晰，语速适中
- 使用质量好的麦克风

💡 **快捷操作**:
- 暂停: 临时停止录音
- 继续: 恢复录音
- 取消: 放弃本次录音

## 🕸️ 知识图谱可视化

### 快速开始

1. **查看图谱**
   ```
   1. 点击侧边栏"知识图谱"
   2. 等待图谱加载完成
   3. 使用鼠标拖拽和滚轮缩放
   4. 点击节点查看详情
   ```

2. **搜索节点**
   ```
   1. 在左侧面板输入搜索关键词
   2. 从搜索结果中选择节点
   3. 图谱自动定位到该节点
   ```

3. **查找路径**
   ```
   1. 在"路径查找"面板选择起点
   2. 选择终点
   3. 点击"查找路径"
   4. 查看所有可能的路径
   5. 点击路径高亮显示
   ```

4. **社区检测**
   ```
   1. 点击"检测社区"按钮
   2. 等待分析完成
   3. 查看社区列表
   4. 点击社区查看成员
   ```

### 功能特性

- ✅ 多种布局算法
- ✅ 节点搜索和筛选
- ✅ 路径查找
- ✅ 社区检测
- ✅ 中心性分析

### 使用技巧

💡 **优化性能**:
- 节点过多时启用"节点聚合"
- 使用筛选功能减少显示节点
- 调整最小权重过滤弱关系

💡 **快捷操作**:
- 双击节点: 打开笔记
- 滚轮: 缩放图谱
- 拖拽: 移动视图
- Ctrl+滚轮: 快速缩放

## 🌐 网页剪藏扩展

### 快速开始

1. **安装扩展**
   ```bash
   # 构建扩展
   cd browser-extension
   npm install
   npm run build:chrome

   # 在Chrome中加载
   1. 打开 chrome://extensions/
   2. 启用"开发者模式"
   3. 点击"加载已解压的扩展程序"
   4. 选择 build/chrome 目录
   ```

2. **剪藏网页**
   ```
   1. 打开要剪藏的网页
   2. 点击扩展图标
   3. 编辑标题和标签
   4. 点击"剪藏到知识库"
   5. 等待保存完成
   ```

3. **批量剪藏**
   ```
   1. 打开多个标签页
   2. 点击扩展图标
   3. 点击"批量剪藏"
   4. 选择要剪藏的标签页
   5. 点击"开始剪藏"
   6. 查看进度和结果
   ```

### 功能特性

- ✅ 智能内容提取
- ✅ AI标签生成
- ✅ AI摘要生成
- ✅ 截图标注
- ✅ 批量剪藏

### 使用技巧

💡 **提高剪藏质量**:
- 使用AI生成标签和摘要
- 添加自定义标签
- 使用截图标注重点内容

💡 **批量操作**:
- 一次选择多个标签页
- 使用搜索过滤标签页
- 查看详细的成功/失败统计

## 🔧 故障排查

### 语音识别不工作

**问题**: 点击麦克风没有反应

**解决方案**:
1. 检查浏览器是否支持Web Speech API
2. 检查麦克风权限是否允许
3. 检查网络连接（在线模式需要）
4. 尝试刷新页面

**问题**: 识别结果不准确

**解决方案**:
1. 在安静环境中录音
2. 说话清晰，语速适中
3. 检查麦克风质量
4. 尝试使用不同的识别引擎

### 图谱渲染缓慢

**问题**: 图谱加载很慢或卡顿

**解决方案**:
1. 启用"节点聚合"
2. 减少显示节点数量
3. 使用筛选功能
4. 清理浏览器缓存
5. 关闭其他占用资源的应用

**问题**: 图谱显示不完整

**解决方案**:
1. 点击"刷新数据"
2. 点击"重建图谱"
3. 检查数据库是否正常
4. 查看控制台错误信息

### 网页剪藏失败

**问题**: 连接失败

**解决方案**:
1. 确保桌面应用正在运行
2. 检查HTTP服务器状态
3. 检查端口23456是否被占用
4. 重启桌面应用

**问题**: 内容提取不完整

**解决方案**:
1. 尝试刷新网页后重新剪藏
2. 检查网页是否有反爬虫机制
3. 手动编辑内容
4. 使用截图功能保存
