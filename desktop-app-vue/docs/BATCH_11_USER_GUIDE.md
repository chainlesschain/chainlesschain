# 第十一批工具使用指南

## 概述

第十一批工具现已集成到ChainlessChain AI系统中，用户可以通过**自然语言对话**直接调用这些前沿科学工具。AI会自动识别用户意图并调用相应的工具。

---

## 如何使用

### 方式1: 自然语言对话（推荐）

直接向AI描述您的需求，AI会自动识别并调用相应的工具。

#### 示例对话：

**用户**: "帮我分析LIGO引力波数据，GPS时间1126259462，持续4秒，使用匹配滤波方法"

**AI**: ✅ 自动调用 `ligo_data_analyzer` 工具，返回应变数据、功率谱密度和触发事件

---

**用户**: "模拟LHC对撞机13 TeV质子对撞产生Higgs玻色子，生成100个事例"

**AI**: ✅ 自动调用 `particle_simulator` 工具，返回粒子事例、截面和运动学信息

---

**用户**: "帮我测定这个木头样品的碳14年龄，Fraction Modern是0.5"

**AI**: ✅ 自动调用 `radiocarbon_dater` 工具，返回C14年龄和校正后的日历年龄

---

### 方式2: 技能选择（精确控制）

在前端界面的"技能管理"页面，可以查看和启用/禁用特定技能：

1. 打开 **技能管理** 页面
2. 浏览第十一批技能（116-125）
3. 点击技能查看详情和包含的工具
4. 启用/禁用特定技能

## 提示词技巧

### ✅ 好的提示词：
1. **明确指定参数**:
   - "分析LIGO数据，探测器选LIGO-Hanford，GPS时间1126259462，持续4秒"

2. **描述具体需求**:
   - "模拟ITER托卡马克，H模运行，等离子体电流15MA，NBI加热33MW"

3. **提供必要上下文**:
   - "这是一个木头样品，用AMS方法测量，Fraction Modern是0.5，δ13C是-25‰，帮我测定C14年龄并校正"

### ❌ 模糊的提示词：
1. "分析一下引力波" → 缺少具体参数
2. "做个粒子模拟" → 未指定对撞机、能量、过程
3. "测个年龄" → 未说明样品类型、测量结果

## 专业术语速查

### 引力波探测
- **GPS时间**: 引力波探测器使用的时间标准
- **应变h**: 引力波引起的空间拉伸/压缩 (h ~ 10^-21)
- **SNR**: 信噪比，通常需要 > 5 才能确认检测
- **Chirp质量**: M_c = (m1·m2)^(3/5) / (m1+m2)^(1/5)

### 粒子物理
- **截面**: 反应概率，单位pb (皮巴恩, 10^-40 m²)
- **LHC**: 大型强子对撞机，能量13 TeV
- **PDF**: 部分子分布函数 (NNPDF, CT18)
- **Higgs**: 质量125.1 GeV，多种衰变道

### 暗物质
- **WIMP**: 弱相互作用大质量粒子 (10-1000 GeV)
- **排除限**: 90% CL上限，σ ~ 10^-45 cm²
- **轴子**: 超轻暗物质候选 (μeV级质量)
- **本底**: radon, cosmogenic, neutron

### 核聚变
- **Q因子**: P_fusion / P_heat，ITER目标Q=10
- **β值**: p_plasma / p_magnetic
- **约束时间**: τ_E，IPB98(y,2)标度律
- **H模**: 高约束模式，优于L模

### 拓扑量子
- **Chern数**: 整数拓扑不变量 (C = 0, ±1, ±2...)
- **Z2不变量**: Z2 = 0 (trivial), 1 (topological)
- **马约拉纳费米子**: 反粒子即自身的费米子
- **量子化电导**: G = 2e²/h = 77.48 μS

### 极地科学
- **δ18O**: 氧同位素比值，‰ VSMOW
- **d-excess**: 氘盈余 = δD - 8·δ18O
- **BP**: Before Present (1950 AD)
- **ka**: 千年 (kilo-annum)

### 考古科技
- **AMS**: 加速器质谱，最灵敏的C14测量方法
- **IntCal20**: 最新国际校正曲线 (2020)
- **cal BP**: 校正后的日历年龄
- **库效应**: 海洋/淡水样品的C14年龄偏老

## 常见问题 (FAQ)

### Q1: 如何查看某个技能包含哪些工具？
**A**: 在"技能管理"页面点击技能卡片，查看详细信息和工具列表。

### Q2: AI会自动选择合适的工具吗？
**A**: 是的，AI会根据用户输入的意图自动匹配最合适的工具。

### Q3: 可以直接调用工具名称吗？
**A**: 可以，例如："使用ligo_data_analyzer分析..."，但推荐使用自然语言描述需求。

### Q4: 工具执行失败怎么办？
**A**: 检查参数是否完整和正确。AI会提示缺少哪些必需参数。

### Q5: 这些工具的计算结果准确吗？
**A**: 工具基于科学公式和算法实现，结果为**模拟估算**，仅供参考。实际科学研究请使用专业软件。
