### 2. 验证功能

启动后，检查以下功能：

- ✅ 模板创建功能正常
- ✅ 技能系统加载（数据库检查）
- ✅ 工具系统加载（数据库检查）

### 3. 数据库验证查询

启动后可通过以下SQL验证数据：

```sql
-- 查询视频技能
SELECT id, display_name, category FROM skills
WHERE category = 'media' OR category = 'content';

-- 查询视频工具
SELECT id, display_name, tool_type, category FROM tools
WHERE category = 'media' OR category = 'content' OR category = 'data';

-- 查询技能-工具映射
SELECT st.id, s.display_name as skill, t.display_name as tool, st.role
FROM skill_tools st
JOIN skills s ON st.skill_id = s.id
JOIN tools t ON st.tool_id = t.id;
```
