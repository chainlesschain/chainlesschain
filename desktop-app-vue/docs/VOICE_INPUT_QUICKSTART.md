# 🎤 语音输入功能 - 快速开始

## 概述

ChainlessChain 桌面应用现已支持语音输入功能！你可以在 AI 聊天界面中使用语音进行输入，无需手动打字。

## 快速开始

### 1. 打开 AI 聊天界面

启动 ChainlessChain 桌面应用，导航到 AI 聊天功能。

### 2. 使用语音输入

在输入框工具栏中，你会看到一个麦克风图标 🎤 按钮：

```
┌─────────────────────────────────────────┐
│  输入框                                  │
│  ┌─────────────────────────────────┐    │
│  │                                 │    │
│  │  给我发消息或描述你的任务...      │    │
│  │                                 │    │
│  └─────────────────────────────────┘    │
│                                         │
│  [@] [📎] [🎤] ← 点击这里开始语音输入    │
└─────────────────────────────────────────┘
```

### 3. 开始录音

1. 点击麦克风按钮
2. 允许浏览器访问麦克风（首次使用时）
3. 录音模态框会弹出
4. 对着麦克风清晰地说话

### 4. 控制录音

录音模态框提供以下控制：

- **暂停** ⏸️: 临时停止录音
- **继续** ▶️: 恢复录音
- **取消** ❌: 放弃本次录音
- **完成** ✅: 结束录音并插入文本

### 5. 查看结果

识别的文本会自动插入到输入框中，你可以：
- 继续编辑文本
- 直接点击"发送"按钮发送消息

## 功能特性

✅ **实时语音识别**: 支持中文语音识别
✅ **连续识别**: 可以连续说话，无需多次点击
✅ **实时预览**: 在录音过程中实时显示识别结果
✅ **录音控制**: 支持暂停、继续、取消操作
✅ **音量指示**: 实时显示录音音量
✅ **时长显示**: 显示录音时长

## 使用提示

💡 **获得最佳效果的建议**:

1. **环境**: 在安静的环境中使用
2. **说话**: 清晰地说话，语速适中
3. **麦克风**: 使用质量较好的麦克风
4. **距离**: 保持适当的麦克风距离（15-30cm）
5. **权限**: 确保已授予麦克风权限

## 常见问题

### Q: 为什么无法使用语音输入？

**A**: 可能的原因：
- 浏览器不支持 Web Speech API
- 麦克风权限未授予
- 麦克风设备未连接

**解决方法**:
1. 使用 Chrome 或 Edge 浏览器
2. 在浏览器设置中允许麦克风权限
3. 检查麦克风设备是否正常工作

### Q: 识别准确度不高怎么办？

**A**: 尝试以下方法：
- 在安静的环境中使用
- 清晰地说话，语速适中
- 使用质量较好的麦克风
- 保持适当的麦克风距离

### Q: 支持哪些语言？

**A**: 当前版本支持中文（zh-CN）。未来版本将支持更多语言。

### Q: 是否需要网络连接？

**A**: 是的，Web Speech API 需要网络连接。未来版本将支持离线语音识别。
