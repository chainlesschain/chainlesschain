### üì¶ Installation

Download the appropriate file for your platform below:

#### Windows (x64)

- `ChainlessChain-win32-x64-0.21.0.zip` - Portable version (recommended for most users)
  - Extract and run `chainlesschain.exe`
  - No installation required

#### macOS

- `ChainlessChain-darwin-x64-0.21.0.dmg` - Intel Mac installer
- `ChainlessChain-darwin-arm64-0.21.0.dmg` - Apple Silicon (M1/M2/M3) installer
  - Open DMG and drag to Applications folder

#### Linux (x64)

- `ChainlessChain-0.21.0.AppImage` - Universal Linux package (recommended)
  - Make executable: `chmod +x ChainlessChain-0.21.0.AppImage`
  - Run: `./ChainlessChain-0.21.0.AppImage`
- `chainlesschain_0.21.0_amd64.deb` - Debian/Ubuntu package
  - Install: `sudo dpkg -i chainlesschain_0.21.0_amd64.deb`
- `chainlesschain-0.21.0.x86_64.rpm` - Fedora/RHEL package
  - Install: `sudo rpm -i chainlesschain-0.21.0.x86_64.rpm`

### üìã System Requirements

#### Minimum Requirements

- **Windows**: Windows 10 (64-bit) or later
- **macOS**: macOS 10.15 Catalina or later
- **Linux**: Ubuntu 20.04 / Fedora 34 / Debian 11 or equivalent (64-bit)
- **RAM**: 4 GB
- **Disk**: 2 GB free space
- **Internet**: Broadband connection for AI services

#### Recommended Requirements

- **RAM**: 8 GB or more
- **Disk**: 10 GB free space (for vector database and models)
- **GPU**: NVIDIA GPU with 8GB+ VRAM (for local LLM acceleration)

#### Backend Services

- **Docker**: Docker Desktop 20.10+ (optional, for backend services)
- **Node.js**: 22.12.0+ (if running from source)
