# 快速入门指南

**欢迎使用 Skill-Tool System!**

这份5分钟快速指南将帮助您快速上手系统的核心功能。

---

## 🎯 5分钟快速上手

### Step 1: 打开技能管理 (30秒)

1. 在左侧导航栏点击"技能管理"
2. 查看系统内置的15个技能
3. 观察页面顶部的统计信息

**你会看到**:
- 📊 总技能数
- ✅ 已启用技能数
- ❌ 已禁用技能数

---

### Step 2: 搜索和筛选 (30秒)

1. 在搜索框输入"代码"
2. 观察筛选结果
3. 点击"分类筛选"下拉菜单
4. 选择"代码开发"分类

**提示**: 可以组合使用搜索和筛选！

---

### Step 3: 查看技能详情 (1分钟)

1. 点击任意技能卡片的"详情"按钮
2. 在侧边抽屉中查看：
   - ✨ 技能基本信息
   - 🛠️ 包含的工具列表
   - 📊 使用统计
   - ⚙️ 配置选项

3. 点击"文档"按钮查看使用说明

---

### Step 4: 创建第一个技能 (2分钟)

1. 点击右上角"创建技能"按钮

2. 填写基本信息：
   ```
   技能名称: 我的第一个技能
   显示名称: My First Skill
   描述: 这是我创建的第一个技能
   分类: 代码开发
   ```

3. 添加标签：`测试`、`学习`

4. 选择关联工具（可选）

5. 点击"保存"

**恭喜！** 你已经创建了第一个技能！

---

### Step 5: 批量操作 (1分钟)

1. 勾选2-3个技能（点击卡片左上角复选框）
2. 观察顶部出现的批量操作栏
3. 点击"批量禁用"按钮
4. 确认操作
5. 点击"批量启用"恢复

**提示**: 删除操作不可恢复，请谨慎使用！

---

### Step 6: 查看统计分析 (30秒)

1. 点击"统计分析"按钮
2. 查看美观的图表：
   - 📈 使用趋势图
   - 🥧 分类分布饼图
   - 🏆 TOP排行榜

---

### Step 7: 浏览工具管理 (30秒)

1. 点击导航栏的"工具管理"
2. 查看系统内置的12个工具
3. 工具以表格形式展示

---

### Step 8: 测试一个工具 (1分钟)

1. 找到"file_reader"工具
2. 点击"测试"按钮
3. 输入参数：
   ```json
   {
     "filePath": "./README.md"
   }
   ```
4. 点击"执行"
5. 查看执行结果

## 🎨 核心概念

### 什么是技能？

技能是**功能的组合**，由多个工具组成。

例如："代码开发"技能包含：
- 📝 file_writer (写入文件)
- 📖 file_reader (读取文件)
- 🎨 code_generator (生成代码)

### 什么是工具？

工具是**基础操作单元**，执行具体任务。

例如：
- `file_reader`: 读取文件内容
- `html_generator`: 生成HTML代码
- `git_init`: 初始化Git仓库

### 技能 vs 工具

| 对比项 | 技能 | 工具 |
|--------|------|------|
| 层级 | 高级 | 基础 |
| 组成 | 多个工具 | 单一功能 |
| 复杂度 | 复杂 | 简单 |
| 示例 | "代码开发" | "file_writer" |

## 💡 最佳实践

### ✅ 建议

1. **先浏览再创建**
   - 查看内置技能和工具
   - 理解现有功能
   - 避免重复创建

2. **合理分类**
   - 使用准确的分类
   - 添加描述性标签
   - 便于后续查找

3. **测试后启用**
   - 创建后先测试
   - 确认无误再启用
   - 避免影响系统

4. **定期清理**
   - 禁用不用的技能
   - 删除重复的工具
   - 保持系统整洁

### ❌ 避免

1. **不要盲目批量删除**
   - 删除不可恢复
   - 先检查依赖关系
   - 建议先禁用

2. **不要过度创建**
   - 避免重复功能
   - 合并相似技能
   - 提高复用率

3. **不要忽略文档**
   - 添加清晰描述
   - 提供使用示例
   - 方便团队协作

## ❓ 遇到问题？

### 常见问题速查

**Q: 搜索无结果？**
- 检查关键词拼写
- 尝试使用标签搜索
- 清空筛选条件

**Q: 无法创建技能？**
- 检查必填字段
- 确保名称唯一
- 查看错误提示

**Q: 批量操作失败？**
- 减少选择数量
- 检查网络连接
- 重试单个操作
