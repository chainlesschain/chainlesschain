## 1. 文档多格式导出

### 功能简介
支持将Markdown文档导出为PDF、Word、HTML、播客脚本、PPT、文章配图等多种格式。

### 使用步骤
1. 在项目详情页面打开任意文档
2. 点击工具栏的"导出"按钮
3. 从下拉菜单选择需要的格式：
   - **下载为 PDF** - 生成PDF文档
   - **下载为 MarkDown** - 复制Markdown文件
   - **下载为 Docx** - 生成Word文档
   - **生成网页** - 生成HTML网页
   - **生成播客** - 生成播客朗读脚本
   - **生成ppt** - 自动生成PPT演示文稿
   - **生成文章配图** - AI分析并生成配图建议

4. 等待处理完成，系统会自动打开文件所在目录

### 注意事项
- PDF/Word导出需要pandoc或puppeteer库（可选）
- PPT生成会根据文档结构自动分页
- 播客脚本会转换为口语化表达
- 文章配图生成AI描述词，需要配合AI绘图工具使用

## 2. PPT自动生成

### 功能简介
从Markdown文档自动生成精美的PowerPoint演示文稿，支持多种主题和自动排版。

### 使用步骤

#### 方法1：通过导出菜单
1. 打开Markdown文档
2. 点击"导出" > "生成ppt"
3. 等待生成完成

#### 方法2：使用AI对话
1. 在项目对话框输入："根据项目文档生成PPT"
2. AI会自动读取文档并生成演示文稿

### PPT特性
- ✅ 自动生成标题页和目录页
- ✅ 根据章节结构自动分页
- ✅ 支持4种主题：商务、学术、创意、深色
- ✅ 自动适配中文字体
- ✅ 包含页码和页眉页脚

### 主题说明
- **商务主题** (business): 蓝色系，适合企业汇报
- **学术主题** (academic): 紫色系，适合学术报告
- **创意主题** (creative): 粉色系，适合创意展示
- **深色主题** (dark): 深色背景，适合炫酷演示

## 3. 项目分享

### 功能简介
将项目分享给他人，支持私密分享和公开访问两种模式。

### 使用步骤
1. 在项目列表页，找到要分享的项目
2. 点击项目操作栏的"分享"按钮
3. 选择分享模式：
   - **仅限自己** - 仅限选定用户访问
   - **公开访问** - 在社区页面公开，任何人都可查看

4. 点击"确定"生成分享链接
5. 使用以下方式分享：
   - **复制链接** - 复制分享URL
   - **微信分享** - 生成二维码（开发中）

### 分享链接格式
```
https://chainlesschain.com/share/[唯一token]
```

### 权限说明
- **私密模式**: 只有持有链接的用户可以访问
- **公开模式**: 项目会在社区页面展示，所有人可见

## 4. 数据可视化

### 功能简介
将CSV数据或结构化数据转换为交互式图表。

### 支持的图表类型
- 📊 **折线图** - 展示趋势变化
- 📊 **柱状图** - 比较数据大小
- 🥧 **饼图** - 展示占比分布
- 📈 **散点图** - 展示数据分布
- 🕸️ **雷达图** - 多维度对比
- 🎯 **漏斗图** - 流程转化分析

### 使用步骤

#### 从CSV文件生成
1. 准备CSV格式的数据文件
2. 在项目对话框输入："将sales.csv可视化为柱状图"
3. AI自动解析数据并生成图表

#### 手动配置
```javascript
// 示例数据
const data = [
  { name: '一月', value: 100 },
  { name: '二月', value: 150 },
  { name: '三月', value: 200 }
];

// 生成图表
const vizEngine = new DataVizEngine();
const chartOption = await vizEngine.generateChartConfig(data, {
  chartType: 'bar',
  title: '月度销售统计',
  theme: 'business'
});
```

### 颜色主题
- `default` - 多彩配色
- `business` - 商务蓝
- `warm` - 暖色调
- `cool` - 冷色调
- `purple` - 紫色系

## 5. AI内容润色

### 功能简介
使用AI改进文章表达，使内容更加专业、流畅。

### 使用场景
- 润色商业文档
- 改进学术论文
- 优化产品说明
- 提升内容质量

### 使用方法

#### 方法1：选中文字润色
1. 在编辑器中选中需要润色的文字
2. 右键菜单选择"AI润色"
3. 选择风格（正式/轻松/专业）
4. 查看润色结果并应用

#### 方法2：通过对话框
1. 在项目对话框输入："帮我润色这段文字：[文字内容]"
2. AI返回润色后的版本
3. 复制结果到文档

### 润色类型
- **内容润色**: 改进表达，保持原意
- **内容扩写**: 增加细节和例子
- **风格调整**: 正式、轻松、专业等

## 6. 播客脚本生成

### 功能简介
将书面文章转换为适合播客朗读的口语化脚本。

### 使用步骤
1. 打开要转换的文档
2. 点击"导出" > "生成播客"
3. AI自动转换为口语化脚本
4. 脚本保存为TXT文件

### 脚本特点
- ✅ 使用第一人称
- ✅ 自然流畅的语言
- ✅ 增加过渡语和互动表达
- ✅ 保持原文核心内容
- ✅ 适合音频传播

## 7. 文章配图生成

### 功能简介
AI分析文章内容，自动生成适合配图的主题和AI绘图描述词。

### 使用步骤
1. 打开文章文档
2. 点击"导出" > "生成文章配图"
3. AI分析文章提取3-5个关键主题
4. 生成包含AI绘图描述的JSON文件

### 输出示例
```json
[
  {
    "title": "工业数据中台架构",
    "description": "现代化的数据中台系统架构图，展示数据采集、清洗、存储、分析和可视化的完整流程，科技感十足，蓝色调为主"
  },
  {
    "title": "实时数据处理",
    "description": "高速流动的数据流，通过管道和处理节点，实时分析和转换，动态效果，紫色和蓝色渐变"
  }
]
```

### 后续操作
1. 查看生成的`themes.json`文件
2. 使用AI绘图工具（Midjourney、DALL-E、Stable Diffusion等）
3. 将描述词输入绘图工具生成图片
4. 将图片添加到文章相应位置

## 常见问题

### Q1: PDF导出失败怎么办？
**A**: PDF导出需要puppeteer库或pandoc工具。如果没有安装，系统会生成HTML文件作为替代，您可以通过浏览器打开HTML并打印为PDF。

### Q2: PPT生成的字体不对？
**A**: PPT默认使用Microsoft YaHei字体。如果您的系统没有该字体，可能会回退到其他字体。您可以在PowerPoint中手动更改字体。

### Q3: 分享链接无法访问？
**A**: 目前分享功能的Web端查看页面还在开发中。分享链接已生成并保存，将来可以通过链接访问项目。

### Q4: 数据可视化支持Excel文件吗？
**A**: 目前仅支持CSV格式。您可以将Excel文件另存为CSV格式后使用。

### Q5: 播客脚本可以直接朗读吗？
**A**: 播客脚本是文本格式，需要配合TTS（文字转语音）工具或人工朗读。未来版本会集成TTS功能。
