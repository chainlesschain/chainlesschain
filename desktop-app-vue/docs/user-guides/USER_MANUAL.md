## 🚀 快速开始

### 第一步：了解系统

Skill-Tool System 是一个技能和工具管理系统，帮助您：
- 📊 管理AI助手的技能集
- 🛠️ 配置和测试各种工具
- 📈 查看使用统计和分析
- 🔗 管理技能与工具的依赖关系

### 第二步：主界面导航

系统包含两个主要模块：

1. **技能管理** (`/skills`)
   - 查看和管理所有技能
   - 创建自定义技能
   - 查看技能统计

2. **工具管理** (`/tools`)
   - 查看和管理所有工具
   - 创建自定义工具
   - 测试工具功能

### 第三步：基本操作

#### 搜索和筛选
1. 使用顶部搜索框输入关键词
2. 点击分类下拉菜单筛选特定类别
3. 点击刷新按钮更新数据

#### 查看详情
1. 点击卡片上的"详情"按钮
2. 在侧边抽屉中查看完整信息
3. 点击"文档"按钮查看使用文档

## 🎯 技能管理

### 查看技能列表

技能以卡片形式展示，每个卡片包含：
- **技能名称**: 技能的显示名称
- **分类标签**: 技能所属类别（代码开发、Web开发等）
- **描述**: 技能的功能说明
- **标签**: 关键词标签
- **使用统计**: 使用次数和成功率
- **状态开关**: 启用/禁用技能

### 创建技能

1. 点击右上角"创建技能"按钮
2. 填写基本信息：
   - **技能名称**: 必填，技能的唯一标识
   - **显示名称**: 可选，用于界面显示
   - **描述**: 技能的功能说明
   - **分类**: 选择技能类别
   - **标签**: 添加关键词标签
3. 配置选项：
   - 选择关联的工具
   - 设置配置参数
   - 上传图标（可选）
4. 点击"保存"完成创建

### 编辑技能

1. 点击技能卡片的"详情"按钮
2. 在详情页面点击"编辑"
3. 修改相关信息
4. 点击"保存"应用更改

### 启用/禁用技能

**单个技能**:
- 点击技能卡片右上角的开关按钮
- 系统会自动保存状态

**批量操作**:
1. 勾选需要操作的技能（左上角复选框）
2. 点击批量操作栏中的"批量启用"或"批量禁用"
3. 确认操作

### 删除技能

**注意**: 删除操作不可恢复！

**单个删除**:
1. 打开技能详情
2. 点击"删除"按钮
3. 确认删除

**批量删除**:
1. 勾选需要删除的技能
2. 点击"批量删除"按钮
3. 查看删除列表
4. 确认删除

## 🛠️ 工具管理

### 查看工具列表

工具以表格形式展示，包含以下列：
- **工具名称**: 工具的标识和描述
- **分类**: 工具类别
- **类型**: 函数/API/命令
- **风险等级**: 1-5级，数字越大风险越高
- **使用情况**: 调用次数和成功率
- **来源**: 内置/插件/自定义
- **状态**: 启用/禁用
- **操作**: 详情/文档/测试

### 创建工具

1. 点击"创建工具"按钮
2. 填写基本信息（Tab 1）:
   - **工具名称**: 唯一标识，创建后不可修改
   - **显示名称**: 界面显示名称
   - **描述**: 工具功能说明
   - **工具类型**: 函数/API/命令
   - **分类**: 选择工具类别
   - **风险等级**: 1-5级

3. 配置参数（Tab 2）:
   - **参数Schema**: JSON格式的参数定义
   - **返回值Schema**: JSON格式的返回值定义
   - **可视化编辑器**: 通过表单添加参数

4. 高级设置（Tab 3）:
   - **权限要求**: 所需的系统权限
   - **配置选项**: JSON格式的配置
   - **使用示例**: 代码示例

5. 点击"保存"完成创建

### 测试工具

1. 点击工具行的"测试"按钮
2. 在测试对话框中输入参数
3. 点击"执行"运行工具
4. 查看执行结果和日志

### 查看工具文档

1. 点击"文档"按钮
2. 在侧边抽屉中查看：
   - 功能描述
   - 参数说明
   - 返回值说明
   - 使用示例
   - 最佳实践

## 📦 批量操作

### 批量选择

**选择单个**:
- 技能：勾选卡片左上角的复选框
- 工具：勾选表格行的复选框

**全选/反选**:
- 点击批量操作栏的复选框
- 全选当前页面/筛选结果

**清空选择**:
- 点击"清空选择"按钮

### 批量启用/禁用

1. 选择多个项目
2. 点击"批量启用"或"批量禁用"
3. 查看确认对话框
4. 点击"确认"执行操作
5. 查看操作结果

### 批量删除

**警告**: 批量删除不可恢复，请谨慎操作！

1. 选择需要删除的项目
2. 点击"批量删除"按钮
3. 查看删除列表（红色显示）
4. 确认操作
5. 查看删除结果

**最佳实践**:
- 删除前先检查依赖关系
- 建议先禁用再删除
- 重要数据先备份

## ❓ 常见问题

### Q1: 如何导入技能？

A: 目前支持两种方式：
1. 手动创建：点击"创建技能"按钮
2. 导入文件：开发中...

### Q2: 技能和工具有什么区别？

A:
- **技能**：高级功能组合，由多个工具组成
- **工具**：基础操作单元，执行具体任务

### Q3: 为什么我的技能无法启用？

A: 可能的原因：
1. 依赖的工具未启用
2. 权限不足
3. 配置错误

解决方法：
1. 检查依赖关系图
2. 启用所需工具
3. 检查技能配置

### Q4: 如何备份我的配置？

A:
1. 导出功能开发中
2. 数据库文件位置：`data/chainlesschain.db`

### Q5: 批量操作失败了怎么办？

A:
1. 查看错误提示
2. 检查单个项目状态
3. 重试失败的项目
4. 联系技术支持

### Q6: 如何查看操作历史？

A:
- 在统计分析中查看使用记录
- 操作日志功能开发中

### Q7: 大数据集加载慢怎么办？

A: 系统已优化大数据集性能：
- 超过50项自动启用虚拟滚动
- 表格支持分页加载
- 建议使用搜索和筛选功能

## 🔧 故障排除

### 问题1: 页面加载失败

**症状**: 空白页面或加载错误

**解决方法**:
1. 刷新页面 (`Ctrl+R`)
2. 清除浏览器缓存
3. 检查网络连接
4. 查看开发者控制台

### 问题2: 搜索无结果

**症状**: 搜索后显示空白

**解决方法**:
1. 检查搜索关键词拼写
2. 尝试其他关键词
3. 清空搜索重试
4. 检查分类筛选

### 问题3: 批量操作无响应

**症状**: 点击批量操作按钮无反应

**解决方法**:
1. 确认已选择项目
2. 检查浏览器控制台错误
3. 刷新页面重试
4. 减少选择数量

### 问题4: 统计图表不显示

**症状**: 统计页面无图表

**解决方法**:
1. 确保有数据
2. 刷新统计页面
3. 检查浏览器兼容性
4. 更新浏览器版本

### 问题5: 权限错误

**症状**: 操作时提示权限不足

**解决方法**:
1. 检查用户权限
2. 联系管理员
3. 重新登录
