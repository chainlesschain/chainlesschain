## 🔍 技术规范

### 模板结构
所有新增模板严格遵循以下规范：

```json
{
  "id": "tpl_{category}_{name}_{number}",
  "name": "template_name",
  "display_name": "中文显示名称",
  "description": "模板描述",
  "icon": "🎯",
  "cover_image": "/templates/covers/xxx.png",
  "category": "分类名称",
  "subcategory": "子分类",
  "tags": ["标签1", "标签2"],
  "project_type": "document",
  "prompt_template": "详细的AI提示词（使用Handlebars语法）",
  "variables_schema": [/* 变量定义 */],
  "file_structure": {/* 文件结构定义 */},
  "default_files": [],
  "is_builtin": true,
  "author": "ChainlessChain Team",
  "version": "1.0.0",
  "usage_count": 0,
  "rating": 0,
  "rating_count": 0,
  "required_skills": ["skill1", "skill2"],
  "required_tools": ["tool1", "tool2"],
  "execution_engine": "document/audio/image/video"
}
```

### 变量类型支持
- `string`: 文本输入
- `number`: 数字输入（支持min/max）
- `select`: 单选下拉框
- `multiselect`: 多选（部分模板）
