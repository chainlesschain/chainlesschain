# 视频项目功能 - 快速入门指南

**版本**: v0.17.0
**更新日期**: 2025-12-30

---

## 🚀 快速开始

### 1. 启动应用

```bash
cd desktop-app-vue
npm run dev
```

数据库迁移会自动执行，新增的15个技能和20个工具将自动加载。

### 2. 创建视频项目

1. 打开应用后，点击"创建新项目"
2. 选择"视频"分类
3. 浏览22个视频模板，选择适合的模板
4. 填写项目信息
5. 点击"创建"

---

## 📚 视频模板分类指南

### 短视频制作 (4个)

**适用平台**: 抖音、快手、视频号、Instagram、TikTok

1. **短视频脚本-抖音/快手**

2. **Instagram Reels脚本**

3. **微信视频号脚本**
   - 用途：1-60分钟视频

4. **TikTok海外脚本**

**快速选择**:

## 🛠️ 技能和工具使用

### 查看已有技能

```sql
-- 查询所有视频相关技能
SELECT display_name, description, category
FROM skills
WHERE category = 'media' OR name LIKE '%video%';
```

### 查看可用工具

```sql
-- 查询所有视频相关工具
SELECT display_name, description, category
FROM tools
WHERE category = 'media';
```

### 技能-工具关联

系统会自动推荐相关工具：

- **视频剪辑技能** → 剪辑软件推荐、时间轴计算器、转场库...
- **字幕制作技能** → 自动字幕生成、字幕样式、时间轴同步...
- **短视频运营技能** → 平台规格、标题生成、标签推荐...

## 💡 使用技巧

### 技巧1: 选择合适的模板

| 如果你想... | 推荐模板 |
|------------|---------|
| 在抖音上爆款 | 短视频脚本 |
| 做知识分享 | B站长视频 / 视频教程 |
| 做产品推广 | 商业广告片 / 产品测评 |
| 记录生活 | Vlog系列 |
| 做深度内容 | 纪录片 / 人物专访 |

### 技巧2: 组合使用

复杂项目可以使用多个模板：

1. **拍摄阶段**: 分镜头脚本表 + 拍摄执行方案
2. **后期阶段**: 视频剪辑提纲 + 视频调色方案 + 视频特效清单
3. **发布阶段**: 多平台适配（抖音+B站+YouTube）

### 技巧3: 自定义修改

所有模板都可以自定义：

1. 创建项目后打开生成的文档
2. 根据实际需求修改内容
3. 保存并继续使用

## 📝 常见问题

### Q1: 模板不显示？

**解决方案**:
1. 确认应用已完全启动
2. 检查数据库迁移是否执行
3. 重启应用

### Q2: 如何添加自定义模板？

**步骤**:
1. 复制现有模板JSON文件
2. 修改内容和变量
3. 放入 `templates/video/` 目录
4. 重启应用自动加载

### Q3: 技能和工具在哪里查看？

技能和工具数据在数据库中，可以通过：
- 项目创建时的智能推荐
- 数据库查询直接访问

### Q4: 支持哪些平台？

**国内**:
- 抖音、快手、B站、视频号、小红书

**海外**:
- YouTube、TikTok、Instagram

**直播**:
- 抖音直播、淘宝直播、快手直播、B站直播

## 🎯 最佳实践

### 1. 短视频创作流程

```
选题策划 → 短视频脚本模板
→ 拍摄
→ 剪辑（使用剪辑提纲）
→ 调色（使用调色方案）
→ 多平台发布
```

### 2. 长视频创作流程

```
主题确定 → 视频教程大纲 / B站视频策划
→ 分镜设计
→ 拍摄执行方案
→ 拍摄
→ 后期制作（剪辑+调色+特效）
→ 发布+运营
```

### 3. 直播运营流程

```
直播脚本策划
→ 准备和预热
→ 执行直播
→ 直播复盘报告
→ 优化下一次
```
