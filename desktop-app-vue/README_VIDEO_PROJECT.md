## 📚 使用流程

### 1️⃣ 创建视频项目

```
应用启动 → 新建项目 → 选择"视频"分类 → 浏览29个模板
```

### 2️⃣ 选择合适模板

**示例：短视频创作**
- 国内平台 → 选择"短视频脚本-抖音/快手"
- 海外平台 → 选择"Instagram Reels"或"TikTok海外脚本"

**示例：长视频创作**
- 知识分享 → 选择"B站长视频策划"或"视频教程大纲"
- 深度内容 → 选择"纪录片策划案"

### 3️⃣ 填写项目信息

根据模板提供的变量填写：
- 基础信息（标题、时长、风格等）
- 可选配置（是否包含字幕、是否需要配音等）
- 平台选择（针对多平台模板）

### 4️⃣ 生成项目文档

系统自动生成：
- 完整的脚本文档
- 拍摄/制作清单
- 预算表格（如适用）
- 相关参考资料

## 💡 使用技巧

### 技巧1：根据目标平台选择模板

| 目标平台 | 推荐模板 |
|----------|----------|
| 抖音爆款 | 短视频脚本-抖音/快手 |
| B站知识区 | B站长视频策划 / 视频教程大纲 |
| YouTube | YouTube长视频 |
| 产品推广 | 商业广告片 / 产品测评 |
| 生活记录 | Vlog系列（旅行/美食/日常） |

### 技巧2：组合使用多个模板

复杂项目可分阶段使用：
1. **前期** - 分镜头脚本表 + 拍摄执行方案
2. **拍摄** - 按照执行方案实施
3. **后期** - 视频剪辑提纲 + 视频调色方案 + 视频特效清单
4. **发布** - 多平台适配（抖音+B站+YouTube）

### 技巧3：利用AI工具提升效率

- 使用"自动字幕生成"节省字幕制作时间
- 使用"视频标题生成器"优化标题吸引力
- 使用"爆款预测器"评估内容潜力

## 🐛 问题排查

### Q1: 模板不显示？
**解决方案**:
1. 确认应用已完全启动
2. 检查数据库迁移是否执行
3. 重启应用

### Q2: 如何添加自定义模板？
**步骤**:
1. 复制现有模板JSON文件
2. 修改内容和变量
3. 放入 `src/main/templates/video/` 目录
4. 重启应用自动加载

### Q3: 数据库迁移失败？
**检查**:
- 删除旧数据库文件（如果是测试环境）
- 检查SQL文件格式
- 查看控制台错误日志
