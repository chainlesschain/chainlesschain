# ChainlessChain 安装信息

欢迎安装 ChainlessChain - 去中心化个人AI管理系统！

## 系统要求

- **操作系统**: Windows 10 (1809+) 或 Windows 11
- **处理器**: 64位处理器
- **内存**: 建议 4GB RAM 或更高
- **硬盘空间**: 至少 1GB 可用空间

## 主要功能

✅ **知识库管理** - 个人第二大脑，支持 Markdown、PDF、Word 等格式
✅ **AI 智能对话** - 集成多种 LLM 模型（本地/云端）
✅ **端到端加密** - P2P 加密通讯，保护隐私
✅ **去中心化身份** - 基于 W3C DID 标准
✅ **硬件级安全** - 支持 U-Key/SIMKey 加密存储
✅ **多端同步** - 跨设备安全同步您的数据

## 安装后操作

安装完成后，首次启动应用时：

1. **选择数据目录** - 选择您的数据存储位置
2. **配置 LLM 服务** - 可选择本地 Ollama 或云端服务
3. **创建身份** - 设置您的去中心化身份
4. **开始使用** - 创建您的第一个笔记！

## 数据存储位置

用户数据将存储在：
```
%APPDATA%\chainlesschain-desktop-vue\
```

包括：
- 知识库数据
- 配置文件
- 加密密钥（如使用 U-Key）
- 同步日志

## 卸载说明

卸载应用时，您可以选择：
- **保留数据** - 重新安装后恢复您的笔记和设置
- **完全删除** - 删除所有应用数据

## 隐私承诺

ChainlessChain 高度重视您的隐私：

- ✅ 所有数据本地存储，您完全掌控
- ✅ 端到端加密通讯
- ✅ 可选择完全离线使用（本地 LLM）
- ✅ 不收集、不上传任何个人数据
