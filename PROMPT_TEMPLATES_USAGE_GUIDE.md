## 🎉 功能说明

您现在可以在**"新建项目"**页面直接使用职业专用Prompt模板了！我们为医生、律师、教师和研究员四大职业群体提供了24个专用模板，点击模板即可自动填充到"需求描述"输入框中。

---

## 📍 功能入口

### 方式一：通过新建项目页面（推荐）

1. **打开Electron应用**
2. **点击左侧菜单** → "项目管理" → "我的项目"
3. **点击"新建项目"按钮**
4. **选择"AI辅助创建"标签页**
5. **查看"职业专用模板"区域** - 就在页面顶部！

### 方式二：通过AI助手页面

1. **打开Electron应用**
2. **点击左侧菜单** → "知识与AI" → "AI对话"
3. **查看Prompt建议面板**
4. **点击"医疗"、"法律"、"教育"或"研究"分类按钮**

## 🚀 快速上手

### 步骤1：打开新建项目页面

```
左侧菜单 → 项目管理 → 我的项目 → 新建项目
```

### 步骤2：选择职业分类

在"职业专用模板"区域，您会看到4个职业按钮：

- 🏥 **医疗** - 7个模板（病历、诊断、用药等）
- ⚖️ **法律** - 7个模板（案件分析、合同审查等）
- 👨‍🏫 **教育** - 7个模板（课程设计、作业批改等）
- 🔬 **研究** - 3个模板（实验设计、数据分析等）

**点击您的职业按钮**，例如点击"医疗"。

### 步骤3：选择模板

点击职业按钮后，会显示该职业的所有Prompt模板卡片。每个卡片显示：
- 模板名称
- 模板描述
- 箭头图标（→）

**点击任意模板卡片**，例如"病历记录助手"。

### 步骤4：查看填充效果

模板内容会**自动填充**到下方的"需求描述"文本框中。

如果模板包含变量（如`{{patientName}}`），会显示为：`[请填写patientName]`，您需要手动替换这些占位符。

### 步骤5：完善并创建项目

1. **检查填充的需求描述**，根据需要修改
2. **填写项目名称**（可选）
3. **选择项目类型**
4. **点击"创建项目"按钮**

## 💡 使用技巧

### 技巧1：模板变量替换

有些模板包含变量（用双花括号包围，如`{{patientName}}`）。这些变量会在填充时显示为占位符：

```
原始模板：
患者姓名：{{patientName}}
性别：{{gender}}
年龄：{{age}}岁

填充后：
患者姓名：[请填写patientName]
性别：[请填写gender]
年龄：[请填写age]岁
```

**您需要手动替换这些占位符**为实际信息：

```
最终内容：
患者姓名：张三
性别：男
年龄：45岁
```

### 技巧2：组合使用多个模板

您可以多次选择不同的模板，将内容追加到需求描述中：

1. 点击"病历记录助手"模板 → 填充基础病历结构
2. 修改内容
3. 点击"诊断辅助分析"模板 → 追加诊断分析需求
4. 整合两部分内容

### 技巧3：自定义修改模板内容

模板内容填充后，您可以**自由编辑**文本框中的内容：

- 删除不需要的部分
- 添加额外的需求
- 调整语言风格
- 补充具体信息

### 技巧4：结合项目类型使用

选择Prompt模板后，记得设置合适的项目类型：

- 医疗/法律模板 → 选择"文档处理"或"数据分析"
- 教育模板 → 选择"文档处理"或"Web开发"
- 研究模板 → 选择"数据分析"

1. **打开新建项目页面**

3. **选择"病历记录助手"模板**

请帮我生成结构化的病历记录：

患者信息：
- 姓名：[请填写patientName]
- 性别：[请填写gender]
- 年龄：[请填写age]岁

[请填写chiefComplaint]

既往史：
[请填写pastHistory]

请以专业、准确的方式记录。

5. **修改占位符**为实际需求:

```
请创建一个电子病历管理系统，包含以下功能：

患者信息管理：
- 基本信息录入（姓名、性别、年龄、联系方式）

病历记录功能：
- 主诉快速录入

- 体格检查记录
- 诊断结果保存

系统要求：
- Web界面，响应式设计
- 支持模板化录入，提高效率
- 数据加密存储
```

6. **填写项目名称**："智能电子病历系统"
7. **选择项目类型**："Web开发"
8. **点击"创建项目"**

## 🎯 常见问题

### Q1: 我没看到"职业专用模板"区域？

**A**: 请确认：
1. 您在"新建项目"页面
2. 选择的是"AI辅助创建"标签页（不是"手动创建"）
3. 应用已经完全启动（等待5-10秒）

如果还是看不到，请刷新页面或重启应用。

### Q2: 点击模板后没有反应？

**A**: 请检查：
1. 打开浏览器开发者工具（F12或Cmd+Option+I）
2. 查看Console标签是否有错误信息
3. 确认`window.electronAPI.promptTemplate`对象存在

可能的原因：
- Preload脚本未正确加载
- 数据库未初始化

**解决方案**：重启应用并等待数据库初始化完成。

### Q3: 模板内容太长，如何简化？

**A**: 模板填充后，您可以自由编辑文本框中的内容：
- 删除不需要的部分
- 简化描述
- 只保留关键需求

### Q4: 我的职业不在列表中怎么办？

**A**: 目前支持4个职业：医疗、法律、教育、研究。

如果您是其他职业：
1. 选择最接近的职业模板作为参考
2. 或直接使用底部的"示例卡片"
3. 或手动编写需求描述

我们计划在未来版本中添加更多职业模板。

### Q5: 模板变量太多，有没有快速填充的方法？

**A**: 当前版本需要手动替换变量占位符。

**临时解决方案**：
1. 复制整个文本到外部编辑器
2. 使用"查找替换"功能批量替换
3. 粘贴回需求描述框

我们计划在未来版本中添加变量表单填写功能。

## 🛠️ 技术实现

### 前端组件

- **组件**: `AIProjectCreator.vue`
- **位置**: `desktop-app-vue/src/renderer/components/projects/`
- **API**: `window.electronAPI.promptTemplate.getAll()`

### 后端支持

- **模板管理器**: `prompt-template-manager.js`
- **位置**: `desktop-app-vue/src/main/prompt/`
- **数据库表**: `prompt_templates`

### 数据流

```
用户点击模板卡片
    ↓
Vue组件调用 fillPromptTemplate()
    ↓
通过IPC调用 window.electronAPI.promptTemplate.fill()
    ↓
主进程读取数据库模板
    ↓
替换变量（如有）
    ↓
返回填充后的文本
    ↓
更新 formData.userPrompt
    ↓
文本框显示内容
```
