## 注意事项

### 1. 页面组件状态
部分页面可能处于以下状态:
- ✅ **完全实现**: 功能完整,可正常使用
- 🚧 **部分实现**: 基础框架完成,部分功能待开发
- 📝 **待开发**: 仅有页面框架,功能未实现

建议在测试时检查每个页面的实现状态。

### 2. 企业版功能
企业版功能标记为 "Pro",可能需要:
- 企业版许可证
- 特定权限配置
- 组织上下文环境

### 3. 后端服务依赖
部分功能需要后端服务支持:
- RSS订阅: 需要RSS解析服务
- 邮件管理: 需要邮件服务配置
- 插件市场: 需要插件仓库服务
- 企业仪表板: 需要数据统计服务

确保相关服务已启动:
```bash
docker-compose up -d
```
