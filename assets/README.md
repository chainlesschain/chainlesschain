# 资源文件目录

本目录包含项目使用的静态资源文件。

## 📁 文件列表

### 图标
- **favicon.ico** - 网站收藏夹图标（16x16, 32x32）
- **logo.png** - 项目Logo图标

## 🎨 使用说明

### favicon.ico
用于：
- 浏览器标签页图标
- 书签图标
- 快捷方式图标

### logo.png
用于：
- 应用程序图标
- 文档封面
- 宣传材料
- README展示

## 📝 注意事项

- 图标文件不应频繁更改，以保持品牌一致性
- 如需更新图标，请确保同时更新所有尺寸版本
- Logo文件建议保持PNG格式以支持透明背景
