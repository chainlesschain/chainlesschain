## 🚀 部署步骤

### 方案1：直接部署到服务器

#### 步骤1：上传文件

```bash
# 使用 scp 上传（Linux/Mac）
scp chainlesschain-website.zip user@server:/path/to/website/

# 或使用 FTP/SFTP 客户端（Windows）
# 推荐工具：FileZilla, WinSCP
```

#### 步骤2：解压文件

```bash
# SSH 登录服务器后
cd /path/to/website/
unzip chainlesschain-website.zip
```

#### 步骤3：配置Web服务器

**Nginx 配置示例** (`/etc/nginx/sites-available/chainlesschain`):

```nginx
server {
    listen 80;
    server_name www.chainlesschain.com chainlesschain.com;

    root /path/to/website;
    index index.html;

    # 启用 gzip 压缩
    gzip on;
    gzip_types text/plain text/css application/json application/javascript text/xml application/xml application/xml+rss text/javascript;

    # 设置缓存
    location ~* \.(jpg|jpeg|png|gif|ico|css|js|svg)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # 处理404错误
    error_page 404 /index.html;

    # 重定向到 HTTPS（如果已配置SSL）
    # return 301 https://$server_name$request_uri;
}
```

#### 步骤4：重启服务

```bash
# Nginx
sudo nginx -t
sudo systemctl reload nginx

# Apache
sudo systemctl reload apache2
```

### 方案2：部署到 GitHub Pages

#### 步骤1：创建仓库

```bash
# 在 GitHub 创建新仓库: chainlesschain.github.io
```

#### 步骤2：上传文件

```bash
# 解压 zip 文件
unzip chainlesschain-website.zip -d website

# 初始化 Git
cd website
git init
git add .
git commit -m "Initial commit: ChainlessChain website"

# 推送到 GitHub
git remote add origin https://github.com/你的用户名/chainlesschain.github.io.git
git branch -M main
git push -u origin main
```

#### 步骤3：启用 GitHub Pages

1. 进入仓库 Settings
2. 找到 Pages 选项
3. Source 选择 `main` 分支
4. 点击 Save

访问: `https://你的用户名.github.io`

## 🔧 故障排查

### 问题1：页面显示乱码

**解决方案**: 确保服务器使用 UTF-8 编码

```nginx
charset utf-8;
```

### 问题2：CSS/JS 文件404

**解决方案**: 检查文件路径是否正确，确保 css/ 和 js/ 目录已上传

### 问题3：导航栏不固定

**解决方案**: 清除浏览器缓存，强制刷新（Ctrl+Shift+R）

### 问题4：手机端显示异常

**解决方案**: 检查 viewport meta 标签，确保响应式设计生效
