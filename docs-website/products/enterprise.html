企业版核心功能

强大而灵活的团队协作能力

多身份架构

一个账号，多个身份（个人、组织A、组织B...）

  • 个人身份：管理个人知识库、私人项目
  • 组织身份：作为团队成员协作
  • 自由切换：一键切换不同身份
  • 完全隔离：每个身份独立数据库
  • 数据加密：AES-256军事级加密

组织管理

任何人都可以创建组织，成为组织Owner

  • 快速创建：5分钟创建你的组织
  • 邀请成员：通过邀请码或DID邀请
  • 角色管理：Owner/Admin/Member/Viewer
  • 权限控制：RBAC权限系统
  • 活动日志：所有操作可追溯
  • 成员管理：添加、移除、角色变更
组织创建流程：

1. 创建组织
   ├─ 填写组织名称、描述
   ├─ 生成组织DID
   ├─ 创建者自动成为Owner
   └─ 生成邀请码

2. 邀请成员
   ├─ 生成6位邀请码
   ├─ 或通过DID直接邀请
   └─ 成员收到通知

3. 成员加入
   ├─ 输入邀请码
   ├─ 验证身份
   ├─ 创建组织身份上下文
   └─ 开始协作

4. 权限管理
   ├─ Owner: 所有权限
   ├─ Admin: 管理权限
   ├─ Member: 读写权限
   └─ Viewer: 只读权限

数据隔离

企业级数据安全保障

  • 独立数据库：每个身份独立DB文件
  • AES-256加密：军事级加密保护
  • 权限分级：细粒度访问控制
  • 审计日志：完整操作记录
  • 数据备份：支持本地备份
  • 隐私保护：数据不离开本地
数据隔离架构：

✓ 独立加密密钥（U-Key派生）

✓ 完全隔离，互不影响
