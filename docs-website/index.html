常见问题

解答您的疑虑

🔐 数据真的安全吗？会不会被黑客攻破？

绝对安全。我们采用U盾/SIMKey物理隔离加密芯片，密钥永不触网，配合SM2国密算法和SQLCipher数据库加密，三重防护。

即使黑客拿到你的电脑，没有U盾也无法解密数据。我们已获得等保三级认证，符合国家信息安全标准。

💰 免费吗？有什么收费项目？

核心功能永久免费。开源MIT协议，基础版完全免费，包括：知识库、本地AI、加密存储、多设备同步。

企业版提供私有化部署、技术支持、定制开发等增值服务，按需付费。

🤖 AI功能需要联网吗？数据会上传到云端吗？

完全本地，无需联网。支持Ollama本地部署大模型（Qwen、LLaMA3等），AI推理全在本地完成。

数据永不上云，100%隐私保护。也可选择云端AI（需联网，数据加密传输）。

📱 支持哪些设备？能跨设备同步吗？

全平台支持。Windows、macOS、Linux、Android、iOS（开发中）。

支持基于Git的端到端加密同步，数据在不同设备间安全传输，中间服务器无法查看。

🛡️ U盾/SIMKey是必需的吗？

开发模式下使用模拟实现，生产环境强烈建议使用真实硬件：

  • 开发/测试：使用模拟模式（任意PIN码：123456）
  • 生产环境：集成真实U盾/SIMKey SDK，获得军事级安全保障
