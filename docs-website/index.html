欢迎使用 ChainlessChain

ChainlessChain（无链之链）是一个基于硬件加密技术的个人AI生态系统，致力于让数据主权回归个人，AI效率触手可及。

🔒

硬件级安全

采用U盾和SIMKey双重物理隔离加密

🤖

本地AI

支持Ollama本地大模型部署

🔄

跨设备同步

基于Git的端到端加密同步

核心特性

  • 数据安全 - SQLCipher数据库加密 + SM2国密算法
  • 隐私保护 - 本地数据存储，完全掌控个人数据
  • AI赋能 - 本地/云端混合AI能力
  • 开源生态 - 核心代码开源，支持插件扩展
  • 全平台 - Windows、macOS、Linux、Android、iOS

快速开始

5分钟快速体验ChainlessChain的核心功能。

桌面端快速开始

1. 安装应用

Bash 
# 克隆仓库
git clone https://github.com/chainlesschain/chainlesschain.git
cd chainlesschain/desktop-app

# 安装依赖
npm install

# 启动应用
npm run dev

2. 首次登录

开发模式PIN码： 123456

首次启动会自动检测U盾（模拟模式），输入PIN码即可登录。

3. 创建第一条笔记

  1. 点击左上角"新建笔记"按钮
  2. 输入标题和内容（支持Markdown）
  3. 按 Ctrl+S 保存或等待自动保存

移动端快速开始

React Native版本

Bash 
# 进入移动端目录
cd mobile-app

# 安装依赖
npm install

# Android
npm run android

# iOS (仅macOS)
cd ios && pod install && cd ..
npm run ios

Android原生版本

Bash 
# 使用Android Studio打开
android-app/

# 或使用命令行构建
cd android-app
./gradlew assembleDebug
💡 提示： 移动端默认使用SIMKey模拟模式，任意4-6位数字PIN码即可登录。

安装部署

系统要求

桌面端

  • 操作系统： Windows 10+, macOS 10.15+, Linux (Ubuntu 20.04+)
  • Node.js： v18.0.0 或更高
  • 内存： 至少 4GB RAM
  • 磁盘空间： 至少 2GB 可用空间

移动端

  • Android： Android 7.0 (API 24) 或更高
  • iOS： iOS 13.0 或更高
  • 开发环境： React Native 0.73+ 或 Android Studio

详细安装步骤

桌面端（Electron）

开发环境

Bash 
cd desktop-app
npm install
npm run dev

生产构建

Bash 
# 构建应用
npm run build

# 打包为安装程序
npm run package

# 输出目录
# Windows: out/make/squirrel.windows/x64/
# macOS: out/make/
# Linux: out/make/

配置AI服务（可选）

如需使用AI对话功能，需要部署Ollama服务：

Bash 
# 使用Docker部署
cd backend/docker

# Linux/Mac
./setup.sh

# Windows
setup.bat

# 或手动启动
docker-compose up -d
⚠️ 注意： AI服务需要较大的系统资源，建议至少8GB内存和10GB磁盘空间。

常见问题

Q: 数据存储在哪里？

A: 所有数据都存储在本地加密数据库中，默认位置：

  • Windows: %APPDATA%/ChainlessChain/
  • macOS: ~/Library/Application Support/ChainlessChain/
  • Linux: ~/.config/ChainlessChain/

Q: 如何备份数据？

A: 建议使用Git同步功能进行数据备份：

  1. 在设置中启用Git同步
  2. 配置远程仓库地址
  3. 数据会自动加密后同步到远程

Q: U盾/SIMKey是必需的吗？

A: 开发模式下使用模拟实现，生产环境建议使用真实硬件：

  • 开发/测试：使用模拟模式（任意PIN码）
  • 生产环境：集成真实U盾/SIMKey SDK

Q: AI功能需要联网吗？

A: 支持两种模式：

  • 本地模式： 使用Ollama部署本地模型，完全离线
  • 云端模式： 连接云端AI服务，需要网络连接
