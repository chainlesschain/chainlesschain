## 📞 需要帮助？

### 常见问题

**Q: APK 安装失败？**
A: 确保手机允许安装来自未知来源的应用，并且有足够的存储空间（至少 200 MB）。

**Q: 应用闪退？**
A: 检查 Android 版本（最低 API 26 / Android 8.0），查看 logcat 日志：

```bash
adb logcat | grep ChainlessChain
```

**Q: 远程控制功能在哪里？**
A: 暂时因为 WebRTC 依赖问题被禁用，首页远程控制按钮显示为半透明。修复方案见 `QUICK_FIX_REMOTE_CONTROL.md`。

**Q: 如何配置 LLM？**
A: 首页 → 点击"LLM设置" → 输入火山引擎或 OpenAI API 密钥 → 测试连接。
