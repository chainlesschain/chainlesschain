## 🔧 关于 Ollama 配置

如果要测试 Ollama 成功连接，需要：

1. **在 PC 上启动 Ollama**:

   ```powershell
   # 设置监听所有接口
   $env:OLLAMA_HOST = "0.0.0.0:11434"
   ollama serve
   ```

2. **获取 PC IP 地址**:

   ```powershell
   ipconfig | findstr IPv4
   # 例如: 192.168.1.100
   ```

3. **在应用中配置**:
   - Base URL: `http://192.168.1.100:11434` (替换为您的 PC IP)
   - 模型: `qwen2.5:latest`

4. **测试连接**:
   - 点击"测试连接"
   - 应该显示"连接成功"
   - 成功消息显示 4秒

**详细指南**: 请查看 `OLLAMA_ANDROID_GUIDE.md`

## 🐛 关于文件浏览器崩溃

如果测试文件浏览功能：

1. **清空日志**:

   ```bash
   adb logcat -c
   ```

2. **点击"文件浏览"触发崩溃**

3. **立即收集日志**:

   ```bash
   adb logcat -d > crash.log
   ```

4. **查看崩溃信息**:
   ```powershell
   Select-String -Path crash.log -Pattern "FATAL|chainlesschain" -Context 10,20
   ```

请将 crash.log 或相关错误信息反馈给我。
