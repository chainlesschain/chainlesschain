# 🚀 ChainlessChain Android v0.30.0 快速开始指南

欢迎使用 ChainlessChain Android v0.30.0！本指南将帮助您在 5 分钟内上手所有新功能。

## 📱 第一步：安装应用

### 方式 1: 从 GitHub 下载
1. 访问 [Releases 页面](https://github.com/yourusername/chainlesschain/releases/tag/v0.30.0)
2. 下载 `chainlesschain-v0.30.0.apk`
3. 在设备上安装（需要允许「未知来源」）

### 方式 2: 从源码构建
```bash
cd android-app
./gradlew assembleDebug
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 👥 添加你的第一个好友

### 通过 DID 搜索

1. **打开应用**，导航到「社交」→「好友」
2. **点击右上角的「添加好友」图标** (➕)
3. **输入好友的 DID**（如 `did:test:friend:123`）
4. **点击「添加」** 按钮
5. **输入验证消息**（可选），点击「发送」
6. ✅ 完成！等待对方接受

### 通过附近的人

1. **打开「添加好友」页面**
2. **滚动到「附近的人」区域**
3. **等待自动扫描**（基于 P2P 网络）
4. **点击想要添加的用户**
5. **发送好友请求**

### 通过推荐好友

1. **打开「添加好友」页面**
2. **滚动到「推荐好友」区域**
3. **查看基于共同好友的推荐**
4. **点击「添加」发送请求**

## 📝 发布你的第一条动态

### 纯文字动态

1. **导航到「动态」页面**
2. **点击右下角的「发布」按钮** (➕)
3. **在输入框中输入内容**（如「我的第一条动态！」）
4. **点击「发布」**
5. ✅ 完成！你的动态已发布到时间流

### 带图片的动态

1. **点击「发布」按钮**
2. **点击「图片」图标** 📷
3. **选择 1-9 张图片**
4. **查看预览网格**（3 列布局）
5. **可选：点击 ❌ 删除不需要的图片**
6. **输入动态内容**
7. **点击「发布」**
8. ✅ 等待上传进度完成，动态已发布！

**小技巧**:
- 图片会自动压缩至 < 5MB
- 最大尺寸 1920×1920，节省流量
- 支持 JPG, PNG, WebP 格式

### 带链接预览的动态

1. **点击「发布」按钮**
2. **输入或粘贴链接**（如 `https://kotlinlang.org`）
3. **等待 0.5 秒**，预览卡片自动加载
4. **验证预览内容**（标题、描述、图片）
5. **可选：点击 ❌ 移除预览**
6. **输入动态内容**
7. **点击「发布」**
8. ✅ 完成！带预览卡片的动态已发布

**支持的网站**:
- ✅ 新闻网站（CNN, BBC, 等）
- ✅ 开发者社区（GitHub, Stack Overflow）
- ✅ 视频平台（YouTube, Vimeo）
- ✅ 社交媒体（Twitter, Facebook）

## 🛡️ 保护你的账号

### 屏蔽用户

**场景**: 不想看到某个用户的内容

1. **打开该用户的动态**
2. **点击右上角「更多」菜单** (⋮)
3. **选择「屏蔽该用户」**
4. **确认屏蔽**
5. ✅ 完成！该用户的所有动态和评论将被隐藏

**管理屏蔽列表**:
1. **打开「设置」**
2. **选择「屏蔽用户」**
3. **查看已屏蔽用户列表**
4. **点击「取消屏蔽」可恢复**

### 举报不当内容

**场景**: 发现垃圾信息、骚扰、不实信息等

1. **打开不当动态**
2. **点击右上角「更多」菜单** (⋮)
3. **选择「举报」**
4. **选择举报原因**:
   - 📧 垃圾信息
   - 🚫 骚扰行为
   - ❌ 不实信息
   - ⚠️ 不当内容
   - ©️ 版权侵权
   - 📝 其他原因
5. **填写详细描述**（可选但推荐）
6. **点击「提交」**
7. ✅ 举报已提交，我们会尽快处理

**举报状态**:
- 待处理 ⏳
- 已审核 ✅
- 已拒绝 ❌

## 💡 使用技巧

### 提升使用体验

1. **搜索好友**
   - 支持搜索昵称、备注名、DID
   - 使用部分关键词即可（如「张」可以找到「张三」）

2. **管理动态**
   - 长按自己的动态可以编辑或删除
   - 置顶重要动态让好友优先看到

3. **节省流量**
   - 图片自动压缩，无需担心流量消耗
   - 链接预览自动缓存，重复访问无额外流量

4. **保护隐私**
   - 备注名仅自己可见
   - 屏蔽用户不会通知对方
   - 举报内容匿名提交

### 性能优化建议

1. **定期清理缓存**
   - 设置 → 存储 → 清理缓存
   - 建议每月清理一次

2. **控制图片数量**
   - 虽然支持 9 张，但 3-5 张效果最佳
   - 减少上传时间和流量消耗

3. **关闭不必要的通知**
   - 设置 → 通知 → 自定义通知类型
   - 保留重要通知（好友请求、@提到）

## ❓ 常见问题

### Q1: 为什么我的图片上传失败？

**A**: 可能的原因：
1. **网络问题** - 检查网络连接，切换 Wi-Fi/移动数据重试
2. **图片过大** - 尝试选择较小的图片（< 10MB）
3. **格式不支持** - 仅支持 JPG, PNG, WebP

### Q2: 链接预览为什么不显示？

**A**: 可能的原因：
1. **网站不支持** - 部分网站未配置 Open Graph 标签
2. **网络超时** - 等待 5 秒后自动失败，可以刷新重试
3. **已移除** - 检查是否误点了关闭按钮

### Q3: 如何撤销已发送的好友请求？

**A**: 目前版本不支持撤销，请耐心等待对方回应或拒绝。

### Q4: 屏蔽用户后对方会知道吗？

**A**: 不会。屏蔽是单向且匿名的，对方不会收到任何通知。

### Q5: 举报后多久会处理？

**A**: 通常 24-48 小时内会审核。严重违规内容会优先处理。

### Q6: 如何备份我的数据？

**A**:
```bash
# 通过 ADB 备份
adb backup -f chainlesschain_backup.ab com.chainlesschain.android

# 恢复备份
adb restore chainlesschain_backup.ab
```
