# Android App 功能访问指南

## 版本：v0.26.2

**更新时间：2026-01-27**

本文档列出了 Android 应用中所有已实现的核心功能及其访问入口，确保所有功能都可以测试和使用。

---

## 应用架构

### 底部导航栏（4个主Tab）

1. **首页** - 功能入口中心
2. **项目** - 项目管理和文件管理
3. **社交** - 去中心化社交网络
4. **我的** - 个人中心和设置

---

## 首页功能入口（3x3 网格）

### 第一行：知识库管理（个人第二大脑）

#### 1. 知识库

- **图标**：📚 Book
- **颜色**：粉红色 (#FF6B9D)
- **功能**：访问知识库列表，管理个人知识库条目
- **导航路由**：`Screen.KnowledgeList`
- **实现状态**：✅ 已实现
- **测试路径**：首页 → 点击"知识库" → 查看/新建/编辑知识条目

#### 2. AI对话

- **图标**：💬 Chat
- **颜色**：绿色 (#4CAF50)
- **功能**：访问AI对话列表，与多个LLM模型进行对话
- **导航路由**：`Screen.ConversationList`
- **实现状态**：✅ 已实现
- **测试路径**：首页 → 点击"AI对话" → 查看对话列表/新建对话/进入聊天

#### 3. LLM设置

- **图标**：⚙️ Settings
- **颜色**：蓝色 (#2196F3)
- **功能**：配置 LLM 服务（火山引擎、OpenAI、本地 Ollama 等）、查看 Token 使用统计
- **导航路由**：`Screen.LLMSettings`
- **实现状态**：✅ 已实现
- **测试路径**：首页 → 点击"LLM设置" → 配置API密钥/查看统计

---

### 第二行：去中心化社交（DID + P2P）

#### 4. 社交广场

- **图标**：🗣️ Forum
- **颜色**：紫色 (#9C27B0)
- **功能**：切换到社交 Tab，查看好友动态、发布内容
- **导航方式**：切换到底部导航第 3 个 Tab
- **实现状态**：✅ 已实现
- **测试路径**：首页 → 点击"社交广场" → 自动切换到社交Tab

#### 5. 我的二维码

- **图标**：🔲 QrCode2
- **颜色**：玫红色 (#E91E63)
- **功能**：展示个人 DID 二维码，供他人扫码添加好友
- **导航路由**：`Screen.MyQRCode`
- **实现状态**：✅ 已实现（v0.31.0+）
- **测试路径**：首页 → 点击"我的二维码" → 查看/保存二维码

#### 6. 扫码添加

- **图标**：📷 QrCodeScanner
- **颜色**：橙色 (#FF9800)
- **功能**：扫描他人二维码添加好友、加入群组
- **导航路由**：`Screen.QRCodeScanner`
- **实现状态**：✅ 已实现（v0.31.0+）
- **测试路径**：首页 → 点击"扫码添加" → 授权相机权限 → 扫描二维码

### 第三行：项目管理 & 数字资产

#### 7. 项目管理

- **图标**：📋 Assignment
- **颜色**：青色 (#00BCD4)
- **功能**：切换到项目 Tab，查看项目列表、项目详情、步骤管理
- **导航方式**：切换到底部导航第 2 个 Tab
- **实现状态**：✅ 已实现
- **测试路径**：首页 → 点击"项目管理" → 自动切换到项目Tab

#### 8. 文件浏览

- **图标**：📁 FolderOpen
- **颜色**：浅绿色 (#8BC34A)
- **功能**：全局文件浏览器，支持项目文件导入、查看、管理
- **导航路由**：`Screen.FileBrowser`
- **实现状态**：✅ 已实现
- **测试路径**：首页 → 点击"文件浏览" → 浏览文件系统/选择项目/导入文件

#### 9. 远程控制

- **图标**：💻 Computer
- **颜色**：深橙色 (#FF5722)
- **功能**：远程控制桌面应用（AI对话、RAG搜索、Agent控制、截图、系统监控）
- **导航路由**：`Screen.RemoteControl`
- **实现状态**：✅ 已实现（Phase 2）

## 其他功能入口

### 从社交 Tab 访问

1. **好友列表** - 查看所有好友，点击进入好友详情
2. **动态列表** - 查看好友发布的动态
3. **发布动态** - 点击右上角"+"按钮发布新动态
4. **动态详情** - 点击动态卡片查看详情、评论、点赞
5. **编辑动态** - 在动态详情页编辑自己的动态
6. **添加好友** - 点击"添加好友"按钮，扫码或输入DID

### 从项目 Tab 访问

1. **项目列表** - 查看所有项目
2. **创建项目** - 点击"+"按钮创建新项目
3. **项目详情** - 点击项目卡片查看详情（V2版本）
4. **步骤管理** - 在项目详情中点击"步骤"查看项目步骤
5. **项目文件** - 在项目详情中点击"文件"浏览项目文件

### 从个人中心 Tab 访问

1. **LLM设置** - 配置 LLM 服务
2. **LLM测试** - 测试 LLM 连接和对话
3. **个人信息** - 查看用户ID、设备ID、生物识别状态
4. **退出登录** - 退出当前账户

### 从首页头像点击访问（ProfileDialog）

1. **个人资料** - 查看用户基本信息
2. **LLM设置** - 快速访问 LLM 设置
3. **LLM测试** - 快速访问 LLM 测试对话
4. **退出登录** - 快速退出登录

## 远程控制子功能

点击首页"远程控制"后，可访问以下子功能：

1. **远程 AI 对话** (`Screen.RemoteAIChat`) - 远程使用桌面端 AI 对话功能
2. **远程 RAG 搜索** (`Screen.RemoteRAGSearch`) - 远程搜索知识库
3. **远程 Agent 控制** (`Screen.RemoteAgentControl`) - 控制桌面端 AI Agent
4. **远程截图** (`Screen.RemoteScreenshot`) - 获取桌面端截图
5. **系统监控** (`Screen.RemoteSystemMonitor`) - 监控桌面端系统状态
6. **命令历史** (`Screen.RemoteCommandHistory`) - 查看远程命令执行历史

## 测试建议

### 基础流程测试

1. **首次启动** → 设置 PIN 码 → 进入首页
2. **知识库测试** → 首页"知识库" → 新建条目 → 编辑 → 保存 → 查看
3. **AI 对话测试** → 首页"AI对话" → 新建对话 → 选择模型 → 发送消息
4. **LLM 设置测试** → 首页"LLM设置" → 配置火山引擎API → 测试连接
5. **社交测试** → 首页"社交广场" → 查看动态 → 发布新动态 → 点赞评论
6. **二维码测试** → 首页"我的二维码" → 查看二维码 → 首页"扫码添加" → 扫描
7. **项目测试** → 首页"项目管理" → 创建项目 → 查看详情 → 添加步骤
8. **文件测试** → 首页"文件浏览" → 浏览文件 → 选择项目 → 导入文件
9. **远程控制测试** → 首页"远程控制" → 连接桌面端 → 测试各子功能

### 导航流程测试

1. **Tab 切换** → 首页 → 项目 → 社交 → 我的 → 返回首页
2. **深度导航** → 首页 → 知识库列表 → 知识编辑器 → 返回 → 返回首页
3. **跨 Tab 导航** → 首页"社交广场" → 自动切换到社交Tab
4. **弹窗导航** → 首页头像 → ProfileDialog → LLM设置 → 返回

## 技术栈

- **UI**: Jetpack Compose + Material Design 3
- **架构**: MVVM + Hilt DI + Navigation Component
- **数据**: SQLCipher + SharedPreferences
- **网络**: Retrofit + OkHttp
- **实时通讯**: WebSocket + libp2p
