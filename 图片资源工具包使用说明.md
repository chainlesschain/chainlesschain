## 📖 详细教程

### Python脚本使用

#### 前提条件
```bash
# 检查Python版本（需要3.6+）
python --version

# 安装依赖
pip install Pillow qrcode[pil]
```

#### 运行脚本
```bash
# 方法1：使用批处理文件（推荐）
生成图片.bat  # Windows
./generate_images.sh  # Mac/Linux

# 方法2：直接运行Python
python generate_images.py
```

#### 自定义配置

**修改二维码链接**：
编辑 `generate_images.py` 第41-46行：
```python
# Android下载二维码
generate_qr_code(
    "https://你的下载链接",  # 修改这里
    os.path.join(QR_DIR, "android-download.png")
)
```

**修改图片尺寸**：
编辑函数调用时的参数：
```python
# 修改二维码尺寸
generate_qr_code(url, filename, size=300)  # 改为300x300

# 修改产品截图尺寸
generate_product_screenshot(type, filename, width=1024, height=768)
```

#### 打开工具
```bash
# 在浏览器中打开
```

## 🔧 故障排除

### 问题1：Python命令找不到

**症状**：
```
'python' 不是内部或外部命令
```

**解决**：
1. 安装Python：https://www.python.org/downloads/
2. 安装时勾选"Add Python to PATH"
3. 重启命令行

---

### 问题2：缺少依赖库

**症状**：
```
ModuleNotFoundError: No module named 'PIL'
```

**解决**：
```bash
pip install Pillow qrcode[pil]
```

---

### 问题3：权限错误（Mac/Linux）

**症状**：
```
Permission denied
```

**解决**：
```bash
# 添加执行权限
chmod +x generate_images.sh
```

---

### 问题4：二维码扫描不出

**检查清单**：
- [ ] 二维码尺寸至少200x200
- [ ] URL格式正确（https://开头）
- [ ] 用手机浏览器打开URL确认可访问
- [ ] 二维码打印/显示清晰

**重新生成**：

## 🌟 进阶技巧

### 批量压缩图片

**使用TinyPNG API**：
```python
# 安装tinify库
pip install tinify

# 批量压缩
import tinify
tinify.key = "你的API Key"

import os
for file in os.listdir("images"):
    if file.endswith(".png"):
        source = tinify.from_file(f"images/{file}")
        source.to_file(f"images/{file}")
```

---

### 自动检测缺失图片

```bash
# 创建check.sh
#!/bin/bash
files=(logo.png images/og-image.png images/qr/android-download.png ...)
for file in "${files[@]}"; do
  [ ! -f "$file" ] && echo "❌ 缺失: $file"
done
```

---

### 生成多尺寸Logo

```python
from PIL import Image

img = Image.open("logo.png")
sizes = [(200, 60), (400, 120), (800, 240)]

for width, height in sizes:
    resized = img.resize((width, height), Image.LANCZOS)
    resized.save(f"logo-{width}x{height}.png")
```
