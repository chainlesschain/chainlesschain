## 🔧 代码结构映射

```swift
SplashView
├── ZStack (容器)
│   ├── LinearGradient (渐变背景)
│   │   └── ignoresSafeArea()
│   │
│   ├── VStack (主内容)
│   │   ├── ZStack (Logo 区域)
│   │   │   ├── Circle (旋转光环)
│   │   │   │   └── rotationEffect(rotationAngle)
│   │   │   │
│   │   │   └── Group (Logo 图标)
│   │   │       ├── if UIImage("LaunchIcon") exists
│   │   │       │   └── Image("LaunchIcon")
│   │   │       └── else
│   │   │           └── ZStack (后备图标)
│   │   │               ├── Circle (背景)
│   │   │               └── Image(systemName: "link.circle.fill")
│   │   │
│   │   ├── VStack (标题区)
│   │   │   ├── Text("ChainlessChain")
│   │   │   └── Text("知识·AI·项目 一体化管理")
│   │   │
│   │   └── VStack (进度区)
│   │       ├── ProgressView()
│   │       └── Text("正在初始化...")
│   │
│   └── VStack (版本信息)
│       └── Text("v0.6.0")
│
└── .onAppear (动画触发)
    ├── withAnimation(.spring) { Logo 动画 }
    ├── withAnimation(.linear.repeatForever) { 旋转 }
    ├── DispatchQueue +0.3s { 标题动画 }
    └── DispatchQueue +0.6s { 进度动画 }
```
