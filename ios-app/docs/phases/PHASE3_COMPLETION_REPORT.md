#### 算法验证

**组合数公式**: C(n, r) = n! / (r! \* (n-r)!)

```
C(5, 2) = 5! / (2! * 3!) = 120 / (2 * 6) = 10 ✅
C(6, 3) = 6! / (3! * 3!) = 720 / (6 * 6) = 20 ✅
C(10, 5) = 10! / (5! * 5!) = 3628800 / (120 * 120) = 252 ✅
```

**排列数公式**: P(n, r) = n! / (n-r)!

```
P(5, 2) = 5! / 3! = 120 / 6 = 20 ✅
```

## 💡 技术亮点

### 1. 排列组合算法实现

```swift
func factorial(_ n: Int) -> Int {
    return n <= 1 ? 1 : n * factorial(n - 1)
}

// 排列数: P(n, r) = n! / (n-r)!
let permutation = factorial(n) / factorial(n - r)

// 组合数: C(n, r) = n! / (r! * (n-r)!)
let combination = factorial(n) / (factorial(r) * factorial(n - r))
```

**优势**:

- 递归实现阶乘函数
- 公式直接映射算法
- 易于理解和维证

### 2. 边界测试覆盖

```swift

```

**验证**:

- ✅ C(5, 0) = 1
- ✅ C(5, 5) = 1
- ✅ C(6, 3) = 20
